# العَدَدُ — Les nombres (من ٣ إلى ١٠)

> [!warning]
> ⚠️ **القاعدة — La règle :**
> 
> العددُ من (**٣ إلى ١٠**) **يُخالِفُ** المعدودَ في التذكير والتأنيث.
> 
> = Le nombre de 3 à 10 est **OPPOSÉ** en genre au nom compté :
> 
> • Si le معدود (nom compté) est **مذكر** → le عدد (nombre) est **مؤنث** (avec ة)
> • Si le معدود (nom compté) est **مؤنث** → le عدد (nombre) est **مذكر** (sans ة)

> [!info]
> **Et le معدود (nom compté) :**
> 
> والمعدودُ يكونُ **جمعًا مجرورًا** بالإضافة (أي يكونُ : **مضافًا إليه**).
> 
> = Le nom compté est au **pluriel** et **مجرور** (car il est مضاف إليه).

---

## أمثلة للمعدود المذكر — Exemples avec un nom masculin

Le singulier du معدود est **مذكر** → le nombre prend **ة** (tā marbūṭa) :

| العدد + المعدود | Traduction | المفرد |
|---|---|---|
| **ثلاثةُ** طلابٍ | trois étudiants | طالب (مذكر) |
| **أربعةُ** رجالٍ | quatre hommes | رجل (مذكر) |
| **خمسةُ** كتبٍ | cinq livres | كتاب (مذكر) |
| **ستةُ** أبوابٍ | six portes | باب (مذكر) |
| عندي **سبعةُ** أقلامٍ | j'ai sept stylos | قلم (مذكر) |
| ثمنُ هذا الكتابِ **ثمانيةُ** ريالاتٍ | le prix de ce livre est huit riyals | ريال (مذكر) |
| لي **تسعةُ** أعمامٍ | j'ai neuf oncles paternels | عمّ (مذكر) |
| في هذا الفصلِ **عشرةُ** طلابٍ | dans cette classe il y a dix étudiants | طالب (مذكر) |

---

## أمثلة للمعدود المؤنث — Exemples avec un nom féminin

Le singulier du معدود est **مؤنث** → le nombre **n'a PAS** de ة :

| العدد + المعدود | Traduction | المفرد |
|---|---|---|
| **ثلاثُ** طالباتٍ | trois étudiantes | طالبة (مؤنث) |
| **أربعُ** نساءٍ | quatre femmes | امرأة (مؤنث) |
| **خمسُ** كلماتٍ | cinq mots | كلمة (مؤنث) |
| **ستُّ** ساعاتٍ | six heures | ساعة (مؤنث) |
| عندي **سبعُ** دراجاتٍ | j'ai sept vélos | دراجة (مؤنث) |
| ثمنُ هذا الكتابِ **ثمانِي** روبيّاتٍ | le prix de ce livre est huit roupies | روبيّة (مؤنث) |
| لي **تسعُ** عمّاتٍ | j'ai neuf tantes paternelles | عمّة (مؤنث) |
| في هذا الفصلِ **عشرُ** طالباتٍ | dans cette classe il y a dix étudiantes | طالبة (مؤنث) |

## الحكم على المفرد — On regarde le singulier !

> [!warning]
> ⚠️ **Attention :**
> 
> الحكمُ في تذكيرِ العددِ وتأنيثِهِ على **المفردِ** في المعدود، و**ليسَ** على الجمعِ.
> 
> = Pour savoir si le nombre prend ة ou pas, on regarde le **singulier** du nom compté, **pas** le pluriel.

| العدد + المعدود | Explication |
|---|---|
| **خمسةُ** كتبٍ | أُنِّثَ العدد (**خمسة**) لأنَّ المفرد في المعدود **مذكر** (**كتاب**) |
| **ثمانيةُ** ريالاتٍ | أُنِّثَ العدد (**ثمانية**) لأنَّ المفرد في المعدود **مذكر** (**ريال**) |

> [!tip]
> 💡 **Astuce :** كتب (pluriel) → كتاب (singulier) = **مذكر** → donc le nombre prend **ة** = خمس**ة** كتبٍ

## 🧠 Résumé

> [!warning]
> ⚠️ **3 règles à retenir :**
> 
> **1.** العدد من ٣ إلى ١٠ **يُخالِف** المعدود في الجنس :
> • معدود مذكر → عدد مؤنث (avec ة) : **ثلاثةُ** طلابٍ
> • معدود مؤنث → عدد مذكر (sans ة) : **ثلاثُ** طالباتٍ
> 
> **2.** المعدود يكون **جمعًا مجرورًا** (مضاف إليه) :
> خمس**ةُ** كتب**ٍ** ← كتبٍ = مضاف إليه مجرور
> 
> **3.** On regarde le **مفرد** (singulier) pour déterminer le genre, pas le جمع (pluriel)
