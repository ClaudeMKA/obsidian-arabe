# قَبْلَ وَبَعْدَ — Avant et Après

**قَبْلَ** (avant) et **بَعْدَ** (après) sont des **ظروف زمان** (adverbes de temps). Ils indiquent **l'ordre dans le temps**.

> [!warning]
> ⚠️ **Règle :**
> 
> **قَبْلَ** et **بَعْدَ** sont des **ظروف** (pas des حروف جر).
> 
> Le mot qui vient après est **مضاف إليه مجرور** (car قَبْلَ / بَعْدَ sont مضاف).

---

## 1️⃣ قَبْلَ — Avant

> [!info]
> **قَبْلَ** = ظرف زمان = **avant**

| Phrase                  | Traduction                             |
|---|---|
| جئتُ **قَبْلَ** الدرسِ       | Je suis venu **avant** le cours        |
| صلِّ **قَبْلَ** النومِ        | Prie **avant** de dormir               |
| اغسلْ يدَيْكَ **قَبْلَ** الأكلِ | Lave tes mains **avant** de manger     |
| وصلتُ **قَبْلَ** المعلمِ     | Je suis arrivé **avant** le professeur |
| **قَبْلَ** سنةٍ             | Il y a un an (**avant** une année)     |

---

## 2️⃣ بَعْدَ — Après

> [!info]
> **بَعْدَ** = ظرف زمان = **après**

| Phrase                  | Traduction                             |
|---|---|
| ذهبتُ **بَعْدَ** الدرسِ      | Je suis parti **après** le cours       |
| نمتُ **بَعْدَ** صلاةِ العشاءِ | J'ai dormi **après** la prière du soir |
| اغسلْ يدَيْكَ **بَعْدَ** الأكلِ | Lave tes mains **après** avoir mangé   |
| جاءَ **بَعْدَ** المعلمِ      | Il est venu **après** le professeur    |
| **بَعْدَ** سنةٍ             | Dans un an (**après** une année)       |

---

## قَبْلَ / بَعْدَ + ضمير — Avec un pronom

| الضمير        | قَبْلَ + ضمير | بَعْدَ + ضمير | Traduction              |
|---|---|---|---|
| ي (moi)       | **قَبْلي**   | **بَعْدي**   | avant moi / après moi   |
| كَ (toi masc.) | **قَبْلَكَ**   | **بَعْدَكَ**   | avant toi / après toi   |
| هُ (lui)       | **قَبْلَهُ**   | **بَعْدَهُ**   | avant lui / après lui   |
| ها (elle)     | **قَبْلَها**  | **بَعْدَها**  | avant elle / après elle |
| نا (nous)     | **قَبْلَنا**  | **بَعْدَنا**  | avant nous / après nous |
| كُم (vous)     | **قَبْلَكُم**  | **بَعْدَكُم**  | avant vous / après vous |
| هُم (eux)      | **قَبْلَهُم**  | **بَعْدَهُم**  | avant eux / après eux   |

### أمثلة

| Phrase                | Traduction                            |
|---|---|
| وصلتُ **قَبْلَكَ**         | Je suis arrivé avant toi              |
| جاءَ **بَعْدَنا**         | Il est venu après nous                |
| مَن جاءَ **قَبْلَهُ** ؟     | Qui est venu avant lui ?              |
| ماذا فعلتَ **بَعْدَها** ؟ | Qu'as-tu fait après elle / après ça ? |

## قَبْلَ / بَعْدَ dans le temps — Expressions courantes

| Expression     | Traduction                 |
|---|---|
| **قَبْلَ** الظهرِ  | avant midi                 |
| **بَعْدَ** الظهرِ  | après-midi                 |
| **قَبْلَ** الفجرِ  | avant l'aube               |
| **بَعْدَ** المغربِ | après le coucher du soleil |
| **قَبْلَ** يومينِ  | il y a deux jours          |
| **بَعْدَ** يومينِ  | dans deux jours            |
| **قَبْلَ** أسبوعٍ  | il y a une semaine         |
| **بَعْدَ** أسبوعٍ  | dans une semaine           |

---

## 🧠 Résumé

|            | قَبْلَ         | بَعْدَ         |
|---|---|---|
| **Sens**   | **avant**   | **après**   |
| **Nature** | ظرف زمان    | ظرف زمان    |
| **+ اسم**  | قَبْلَ الدرسِ   | بَعْدَ الدرسِ   |
| **+ ضمير** | قَبْلي / قَبْلَكَ | بَعْدي / بَعْدَكَ |

> [!tip]
> 💡 **À retenir :**
> 
> **1.** قَبْلَ et بَعْدَ sont des **ظروف زمان**
> **2.** Le mot après est **مضاف إليه مجرور**
> **3.** قَبْلَ الظهرِ = avant midi / بَعْدَ الظهرِ = après-midi
