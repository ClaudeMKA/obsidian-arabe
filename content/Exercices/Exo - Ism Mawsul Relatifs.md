## Exercice 1 — Choisis le bon pronom relatif

Complète avec **الَّذي / الَّتي / اللَّذانِ / اللَّتانِ / الَّذينَ / اللَّاتي** :

| # | Phrase | Pronom relatif |
|---|---|---|
| 1 | جاءَ الرجلُ ... نجحَ | |
| 2 | هذه المرأةُ ... تعملُ هنا | |
| 3 | الطالبانِ ... درسا نجحا | |
| 4 | الطالباتُ ... نجحنَ مجتهداتٌ | |
| 5 | الرجالُ ... جاؤوا أطباءُ | |
| 6 | الطالبتانِ ... درستا نجحتا | |

> [!tip]- Correction
> 1. **الَّذي** (masc. sing.)
> 2. **الَّتي** (fém. sing.)
> 3. **اللَّذانِ** (duel masc.)
> 4. **اللَّاتي** (plur. fém.)
> 5. **الَّذينَ** (plur. masc.)
> 6. **اللَّتانِ** (duel fém.)

## Exercice 2 — Trouve le عائد (pronom de rappel)

Dans chaque phrase, identifie le **عائد** (pronom de rappel dans la صلة الموصول) :

| # | Phrase | Le عائد |
|---|---|---|
| 1 | قرأتُ الكتابَ الَّذي اشتريتَ**هُ** | |
| 2 | هذه المدرسةُ الَّتي درستُ في**ها** | |
| 3 | جاءَ الطالبُ الَّذي نجحَ | |
| 4 | الرجالُ الَّذينَ رأيتُ**هُم** | |

> [!tip]- Correction
> 1. **هُ** — pronom attaché au verbe اشتريتَ (tu l'as acheté)
> 2. **ها** — pronom attaché à في (dans laquelle)
> 3. **ضمير مستتر** (هو) — caché dans le verbe نجحَ (il a réussi)
> 4. **هُم** — pronom attaché au verbe رأيتُ (je les ai vus)

## Exercice 4 — Piège : le duel au منصوب/مجرور

Mets la bonne forme du pronom relatif :

| # | Phrase | اللَّذانِ ou اللَّذَيْنِ ? |
|---|---|---|
| 1 | جاءَ الطالبانِ ... درسا | |
| 2 | رأيتُ الطالبَيْنِ ... درسا | |
| 3 | سلّمتُ على الطالبَيْنِ ... درسا | |

> [!tip]- Correction
> 1. **اللَّذانِ** — مرفوع (sujet) → forme en انِ
> 2. **اللَّذَيْنِ** — منصوب (après رأيتُ) → forme en يْنِ
> 3. **اللَّذَيْنِ** — مجرور (après على) → forme en يْنِ
