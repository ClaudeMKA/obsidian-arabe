## Exercice 1 — Classifie le mot (اسم / فعل / حرف)

Pour chaque mot, indique s'il s'agit d'un **اسم**, **فعل** ou **حرف** :

| # | Mot | Ta réponse |
|---|---|---|
| 1 | كتاب | |
| 2 | كَتَبَ | |
| 3 | في | |
| 4 | جميل | |
| 5 | يَذهبُ | |
| 6 | من | |
| 7 | هو | |
| 8 | اُكتُبْ | |
| 9 | إنَّ | |
| 10 | مدرسة | |

> [!tip]- Correction
> 1. كتاب → **اسم** (sens en lui-même, pas lié au temps)
> 2. كَتَبَ → **فعل** (il a écrit — lié au passé)
> 3. في → **حرف** (pas de sens seul = dans)
> 4. جميل → **اسم** (beau — les adjectifs sont des اسم !)
> 5. يَذهبُ → **فعل** (il va — lié au présent)
> 6. من → **حرف** (de/depuis)
> 7. هو → **اسم** (les pronoms sont des اسم !)
> 8. اُكتُبْ → **فعل** (écris ! — lié au futur/ordre)
> 9. إنَّ → **حرف** (certes — pas de sens seul)
> 10. مدرسة → **اسم** (école)

## Exercice 2 — معرفة ou نكرة ?

Indique si chaque mot est **معرفة** (déterminé) ou **نكرة** (indéterminé) :

| # | Mot | معرفة ou نكرة ? | Pourquoi ? |
|---|---|---|---|
| 1 | كتابٌ | | |
| 2 | الكتابُ | | |
| 3 | محمد | | |
| 4 | هذا | | |
| 5 | رجلٌ | | |
| 6 | كتابُ الطالبِ | | |
| 7 | أنا | | |
| 8 | بيتٌ | | |

> [!tip]- Correction
> 1. كتابٌ → **نكرة** (tanwīn = indéterminé, on peut dire الكتاب)
> 2. الكتابُ → **معرفة** (a الـ)
> 3. محمد → **معرفة** (nom propre = عَلَم)
> 4. هذا → **معرفة** (démonstratif = اسم الإشارة)
> 5. رجلٌ → **نكرة** (tanwīn, on peut dire الرجل)
> 6. كتابُ الطالبِ → **معرفة** (annexion avec un mot défini = مضاف)
> 7. أنا → **معرفة** (pronom = ضمير)
> 8. بيتٌ → **نكرة** (tanwīn)

## Exercice 3 — Quel est le إعراب ?

Donne l'état grammatical (**مرفوع / منصوب / مجرور**) du mot en **gras** :

| # | Phrase | إعراب du mot en gras |
|---|---|---|
| 1 | جاءَ **الطالبُ** | |
| 2 | قرأتُ **الكتابَ** | |
| 3 | في **البيتِ** | |
| 4 | **الجوُّ** جميلٌ | |
| 5 | إنَّ **الولدَ** ذكيٌّ | |
| 6 | كتابُ **الطالبِ** | |
| 7 | رأيتُ **رجُلَيْنِ** | |
| 8 | جاءَ **أبوكَ** | |

> [!tip]- Correction
> 1. الطالبُ → **مرفوع** (فاعل — sujet du verbe)
> 2. الكتابَ → **منصوب** (مفعول به — complément d'objet)
> 3. البيتِ → **مجرور** (après حرف جر في)
> 4. الجوُّ → **مرفوع** (مبتدأ — sujet de phrase nominale)
> 5. الولدَ → **منصوب** (اسم إنَّ)
> 6. الطالبِ → **مجرور** (مضاف إليه)
> 7. رجُلَيْنِ → **منصوب** (مثنى منصوب بالياء)
> 8. أبوكَ → **مرفوع** (أسماء خمسة مرفوع بالواو)

## Exercice 5 — Piège ! Vrai ou Faux ?

| # | Affirmation | V/F |
|---|---|---|
| 1 | Les adjectifs (جميل, كبير) sont des فعل | |
| 2 | Le tanwīn disparaît quand on ajoute الـ | |
| 3 | Les pronoms (هو, أنا) sont des حرف | |
| 4 | Un mot مجرور finit toujours par كسرة | |
| 5 | Le mot après حرف جر est toujours مجرور | |
| 6 | المثنى est مرفوع avec الألف et منصوب avec الياء | |

> [!tip]- Correction
> 1. **FAUX** — Les adjectifs sont des **اسم** !
> 2. **VRAI** — كتابٌ → الكتابُ
> 3. **FAUX** — Les pronoms sont des **اسم** (catégorie معرفة)
> 4. **FAUX** — الممنوع من الصرف prend **فتحة** au مجرور, et les أسماء خمسة prennent **ياء**
> 5. **VRAI** — C'est la règle fondamentale de حروف الجر
> 6. **VRAI** — رجلانِ (مرفوع) → رجلَيْنِ (منصوب/مجرور)
