## Exercice 1 — Analyse grammaticale complète

Donne le **إعراب** et la **raison** pour chaque mot en gras :

| # | Phrase | Mot | إعراب | Raison |
|---|---|---|---|---|
| 1 | هذا **الكتابُ** **الجديدُ** مفيدٌ | الكتابُ | | |
| 2 | | الجديدُ | | |
| 3 | قرأتُ هذا **الكتابَ** **الجديدَ** | الكتابَ | | |
| 4 | | الجديدَ | | |
| 5 | في هذا **الكتابِ** **الجديدِ** | الكتابِ | | |
| 6 | | الجديدِ | | |
| 7 | جاءَ **أبوكَ** | أبوكَ | | |
| 8 | سلّمتُ على **فاطمةَ** | فاطمةَ | | |
| 9 | عِندي **ثلاثةُ** **كتبٍ** | كتبٍ | | |

> [!tip]- Correction
> 1. الكتابُ → **بدل مرفوع** بالضمة (بدل من هذا = مبتدأ)
> 2. الجديدُ → **نعت مرفوع** بالضمة (يتبع المنعوت الكتابُ)
> 3. الكتابَ → **بدل منصوب** بالفتحة (بدل من هذا = مفعول به)
> 4. الجديدَ → **نعت منصوب** بالفتحة
> 5. الكتابِ → **بدل مجرور** بالكسرة (بدل من هذا = مجرور بـ في)
> 6. الجديدِ → **نعت مجرور** بالكسرة
> 7. أبوكَ → **فاعل مرفوع** بال**واو** (أسماء خمسة)
> 8. فاطمةَ → **مجرور** بال**فتحة** (ممنوع من الصرف — عَلَم مؤنث)
> 9. كتبٍ → **مضاف إليه مجرور** بالكسرة (بعد العدد = جمع مجرور)

## Exercice 2 — Corrige les erreurs

Chaque phrase contient **une erreur**. Trouve-la et corrige :

| # | Phrase fausse | L'erreur | Correction |
|---|---|---|---|
| 1 | هؤلاءِ كتبٌ | | |
| 2 | الطالبُ مجتهدُ | | |
| 3 | رأيتُ أبوكَ | | |
| 4 | ثلاثُ رجالٍ | | |
| 5 | سلّمتُ على أحمدِ | | |
| 6 | في كتابٍ أحمرِ | | |
| 7 | هل هذا ؟ | | |
| 8 | الرجلُ الَّتي جاءَ | | |

> [!tip]- Correction
> 1. هؤلاءِ → **هذه** كتبٌ (كتب = غير عاقل → مفرد مؤنث)
> 2. مجتهدُ → **مجتهدٌ** (خبر نكرة → doit avoir le tanwīn)
> 3. أبوكَ → **أباكَ** (منصوب = الألف, pas الواو)
> 4. ثلاثُ → **ثلاثةُ** رجالٍ (رجل = مذكر → nombre avec ة)
> 5. أحمدِ → **أحمدَ** (ممنوع من الصرف → فتحة au مجرور)
> 6. أحمرِ → **أحمرَ** (ممنوع من الصرف → فتحة au مجرور)
> 7. C'est incomplet → **مَن** هذا ؟ ou **ما** هذا ؟ (هل seul ne suffit pas, il faut un اسم استفهام ou ajouter un خبر)
> 8. الَّتي → **الَّذي** جاءَ (الرجلُ = مذكر → الَّذي)

## Exercice 3 — Construis la phrase

Avec les éléments donnés, construis la phrase arabe correcte :

| # | Éléments | Ta phrase |
|---|---|---|
| 1 | démonstratif proche + voiture + neuve + (phrase nominale) | |
| 2 | j'ai lu + ce + livre + rouge | |
| 3 | l'homme + qui + est venu + est médecin | |
| 4 | chez moi + 5 + livres | |
| 5 | je suis allé + avec + mon père + avant + le cours | |

> [!tip]- Correction
> 1. **هذه السيارةُ جديدةٌ** (هذه car مؤنث, السيارةُ = بدل, جديدةٌ = خبر نكرة)
> 2. **قرأتُ هذا الكتابَ الأحمرَ** (tout منصوب, الأحمرَ = نعت)
> 3. **الرجلُ الَّذي جاءَ طبيبٌ** (الَّذي car مذكر, جاءَ = صلة الموصول)
> 4. **عِندي خمسةُ كتبٍ** (كتاب = مذكر → خمسة avec ة)
> 5. **ذهبتُ مَعَ أبي قَبْلَ الدرسِ** (أبي car مضاف à ياء المتكلم → forme normale)

## Exercice 4 — QCM Rapide (20 questions)

Choisis la bonne réponse :

**1.** Les adjectifs en arabe sont des :
- [ ] a) فعل
- [ ] b) حرف
- [ ] c) اسم

**2.** Le tanwīn indique que le mot est :
- [ ] a) معرفة
- [ ] b) نكرة
- [ ] c) مجرور

**3.** Le mot après un حرف جر est :
- [ ] a) مرفوع
- [ ] b) منصوب
- [ ] c) مجرور

**4.** عِندَ est un :
- [ ] a) حرف جر
- [ ] b) ظرف
- [ ] c) فعل

**5.** Le pluriel غير عاقل est traité comme :
- [ ] a) جمع مذكر
- [ ] b) مفرد مؤنث
- [ ] c) مثنى

**6.** Les أسماء خمسة au مرفوع prennent :
- [ ] a) ضمة
- [ ] b) واو
- [ ] c) ألف

**7.** Le ممنوع من الصرف au مجرور prend :
- [ ] a) كسرة
- [ ] b) فتحة
- [ ] c) ضمة

**8.** Le nombre 5 avec كتب (كتاب = مذكر) :
- [ ] a) خمسُ كتبٍ
- [ ] b) خمسةُ كتبٍ
- [ ] c) خمسةُ كتبٌ

**9.** هل est un :
- [ ] a) اسم استفهام
- [ ] b) حرف استفهام
- [ ] c) ظرف

**10.** Le بدل suit le اسم الإشارة dans :
- [ ] a) le genre seulement
- [ ] b) le إعراب seulement
- [ ] c) les 4 accords

> [!tip]- Correction
> 1. **c) اسم**
> 2. **b) نكرة**
> 3. **c) مجرور**
> 4. **b) ظرف**
> 5. **b) مفرد مؤنث**
> 6. **b) واو**
> 7. **b) فتحة**
> 8. **b) خمسةُ كتبٍ** (مذكر → avec ة, مجرور)
> 9. **b) حرف استفهام**
> 10. **b) le إعراب seulement**
