## Exercice 1 — ممنوع من الصرف ou pas ?

Pour chaque mot, dis s'il est ممنوع من الصرف et pourquoi :

| # | Mot | ممنوع ? | Raison |
|---|---|---|---|
| 1 | أحمدُ | | |
| 2 | كتابٌ | | |
| 3 | فاطمةُ | | |
| 4 | أحمرُ | | |
| 5 | مساجدُ | | |
| 6 | رجلٌ | | |

| 8 | صحراءُ | | |
| 9 | إبراهيمُ | | |
| 10 | كُبرى | | |

> [!tip]- Correction
> 1. **ممنوع** — عَلَم (nom propre)
> 2. **Non** — nom ordinaire, accepte le tanwīn : كتابٌ
> 3. **ممنوع** — عَلَم مؤنث (nom propre féminin)
> 4. **ممنوع** — صفة sur le schème أَفْعَل (couleur)
> 5. **ممنوع** — صيغة منتهى الجموع (pluriel brisé : مَفاعِل)
> 6. **Non** — nom ordinaire
> 7. **ممنوع** — عَلَم se terminant par ان
> 8. **ممنوع** — ألف التأنيث الممدودة (اء)
> 9. **ممنوع** — عَلَم أعجمي (nom propre étranger)
> 10. **ممنوع** — ألف التأنيث المقصورة (ى)

## Exercice 2 — Quel est le signe du مجرور ?

Le grand piège : فتحة ou كسرة ?

| # | Phrase | Signe du مجرور | Pourquoi ? |
|---|---|---|---|
| 1 | سلّمتُ على أحمد... | | |
| 2 | في المدرسة... | | |
| 3 | في مساجد... كثيرة | | |
| 4 | في **ال**مساجد... | | |
| 5 | صلّيتُ في مساجد... المدينة... | | |
| 6 | في كتابٍ أحمر... | | |

> [!tip]- Correction
> 1. أحمد**َ** — **فتحة** (ممنوع من الصرف)
> 2. المدرسة**ِ** — **كسرة** (mot normal avec ال)
> 3. مساجد**َ** — **فتحة** (ممنوع من الصرف, صيغة منتهى الجموع)
> 4. المساجد**ِ** — **كسرة** ! (avec **ال** → la كسرة revient !)
> 5. مساجد**ِ** المدينة**ِ** — **كسرة** ! (en état de **مضاف** → la كسرة revient !)
> 6. أحمر**َ** — **فتحة** (ممنوع من الصرف, schème أَفْعَل)

## Exercice 3 — L'exception cruciale : ال et الإضافة

Vrai ou Faux :

| # | Affirmation | V/F |
|---|---|---|
| 1 | مساجدُ n'accepte jamais le tanwīn | |
| 2 | مساجدُ prend toujours فتحة au مجرور | |
| 3 | المساجدِ prend كسرة au مجرور | |
| 4 | مساجدِ المدينةِ prend كسرة au مجرور | |
| 5 | أحمدَ prend toujours فتحة au مجرور | |

> [!tip]- Correction
> 1. **VRAI** — ممنوع من الصرف = jamais de tanwīn
> 2. **FAUX** — il prend فتحة SAUF quand il a **ال** ou qu'il est **مضاف**
> 3. **VRAI** — avec ال la كسرة revient
> 4. **VRAI** — en إضافة la كسرة revient
> 5. **VRAI** — أحمد est un عَلَم, même avec ال ou إضافة c'est فتحة (les noms propres restent ممنوع)
