# Phrases Complexes — تَرْجِمْ مَعَ عِدَّةِ قَوَاعِدَ

> [!warning]
> Chaque phrase combine **plusieurs règles**. Traduis sur papier puis vérifie. Les règles testées sont indiquées entre crochets.

| 2   | Ce livre est à ton père.                               | ,                                                     |

| 7   | Cette voiture est à Ibrahim.                           | [اسْمُ الْإِشَارَةِ + لِـ + مَمْنُوعٌ مِنَ الصَّرْفِ] |
| 8   | Nous sommes allés avant eux en voiture.                | [قَبْلَ + ضَمِير + بِـ]                               |

| 10  | Ces deux livres sont à nous.                           | [اسْمُ الْإِشَارَةِ + الْمُثَنَّى + لِـ]              |
| 11  | N'écris pas avec le stylo rouge !                      | [النَّهْيُ + بِـ + النَّعْتُ]                         |
| 12  | Elle a trois belles voitures.                          | [عِنْدَ + الْعَدَدُ + النَّعْتُ]                      |
| 13  | L'homme qui est venu est mon père.                     | [الِاسْمُ الْمَوْصُولُ + الْأَسْمَاءُ الْخَمْسَةُ]    |
| 14  | J'ai mangé avec la cuillère avant le cours.            | [بِـ + قَبْلَ]                                        |
| 15  | Où est le livre de Mohammed ?                          | [اسْمُ الِاسْتِفْهَامِ + الْإِضَافَةُ]                |

> [!tip]- Correction
> 1. سَلَّمْتُ عَلَى أَحْمَدَ عِنْدَ الْمُعَلِّمِ.
> (**أَحْمَدَ** = مَمْنُوعٌ مِنَ الصَّرْفِ → فَتْحَةٌ بَدَلَ كَسْرَةٍ / **عِنْدَ** = ظَرْفٌ + مُضَافٌ إِلَيْهِ)
>
> 2. هَذَا الْكِتَابُ لِأَبِيكَ.
> (**هَذَا** = اسْمُ إِشَارَةٍ / **لِـ** = حَرْفُ جَرٍّ / **أَبِيكَ** = مَجْرُورٌ بِالْيَاءِ مِنَ الْأَسْمَاءِ الْخَمْسَةِ)
>
> 3. ذَهَبَتِ الطَّالِبَتَانِ مَعَ مُعَلِّمَتِهِمَا.
> (**الطَّالِبَتَانِ** = فَاعِلٌ مَرْفُوعٌ بِالْأَلِفِ مُثَنَّى / **مَعَ** = ظَرْفٌ / **مُعَلِّمَتِهِمَا** = مُضَافٌ إِلَيْهِ + ضَمِيرُ الْمُثَنَّى)
>
> 4. هَلْ عِنْدَكَ كِتَابٌ أَحْمَرُ ؟
> (**عِنْدَكَ** = ظَرْفٌ + ضَمِيرٌ / **أَحْمَرُ** = نَعْتٌ مَرْفُوعٌ بِالضَّمَّةِ بِدُونِ تَنْوِينٍ لِأَنَّهُ مَمْنُوعٌ مِنَ الصَّرْفِ)
>
> 5. كَتَبَ الطَّالِبُ الْمُجْتَهِدُ بِالْقَلَمِ.
> (**الْمُجْتَهِدُ** = نَعْتٌ مَرْفُوعٌ يَتْبَعُ الْمَنْعُوتَ / **بِالْقَلَمِ** = جَارٌّ وَمَجْرُورٌ)
>
> 6. خَرَجَتْ فَاطِمَةُ مِنَ الْمَسْجِدِ الْكَبِيرِ.
> (**فَاطِمَةُ** = مَمْنُوعٌ مِنَ الصَّرْفِ لَكِنْ هُنَا فَاعِلٌ مَرْفُوعٌ بِالضَّمَّةِ / **الْكَبِيرِ** = نَعْتٌ مَجْرُورٌ)
>
> 7. هَذِهِ السَّيَّارَةُ لِإِبْرَاهِيمَ.
> (**هَذِهِ** = مُؤَنَّثٌ / **لِإِبْرَاهِيمَ** = مَمْنُوعٌ مِنَ الصَّرْفِ → فَتْحَةٌ)
>
> 8. ذَهَبْنَا قَبْلَهُمْ بِالسَّيَّارَةِ.
> (**قَبْلَهُمْ** = ظَرْفٌ + ضَمِيرٌ / **بِالسَّيَّارَةِ** = وَسِيلَةُ نَقْلٍ)
>
> 9. رَأَيْتُ أَخَاكَ فِي السُّوقِ.
> (**أَخَاكَ** = مَفْعُولٌ بِهِ مَنْصُوبٌ بِالْأَلِفِ مِنَ الْأَسْمَاءِ الْخَمْسَةِ)
>
> 10. هَذَانِ الْكِتَابَانِ لَنَا.
> (**هَذَانِ** = مُثَنَّى الْإِشَارَةِ / **الْكِتَابَانِ** = مَرْفُوعٌ بِالْأَلِفِ / **لَنَا** = لِـ + ضَمِيرٌ)
>
> 11. لَا تَكْتُبْ بِالْقَلَمِ الْأَحْمَرِ !
> (**لَا تَكْتُبْ** = نَهْيٌ مَجْزُومٌ بِالسُّكُونِ / **الْأَحْمَرِ** = نَعْتٌ مَجْرُورٌ)
>
> 12. عِنْدَهَا ثَلَاثُ سَيَّارَاتٍ جَمِيلَةٍ.
> (**ثَلَاثُ** = بِدُونِ تَاءٍ لِأَنَّ سَيَّارَة مُؤَنَّثٌ → الْعَدَدُ يُخَالِفُ / **جَمِيلَةٍ** = نَعْتٌ مَجْرُورٌ)
>
> 13. الرَّجُلُ الَّذِي جَاءَ أَبِي.
> (**الَّذِي جَاءَ** = صِلَةُ الْمَوْصُولِ / **أَبِي** = خَبَرٌ مَرْفُوعٌ... لَكِنْ هُنَا مُضَافٌ إِلَى يَاءِ الْمُتَكَلِّمِ → لَا يُعْرَبُ بِالْحُرُوفِ)
>
> 14. أَكَلْتُ بِالْمِلْعَقَةِ قَبْلَ الدَّرْسِ.
> (**بِالْمِلْعَقَةِ** = أَدَاةٌ / **قَبْلَ** = ظَرْفُ زَمَانٍ + مُضَافٌ إِلَيْهِ)
>
> 15. أَيْنَ كِتَابُ مُحَمَّدٍ ؟
> (**أَيْنَ** = اسْمُ اسْتِفْهَامٍ لِلْمَكَانِ / **كِتَابُ مُحَمَّدٍ** = مُضَافٌ وَمُضَافٌ إِلَيْهِ)

## Niveau 2 — 3 règles par phrase

| # | Traduis en arabe | Règles |
|---|---|---|
| 16 | Le père d'Ahmed est allé aux mosquées de la ville en voiture. | [الْأَسْمَاءُ الْخَمْسَةُ + مَمْنُوعٌ مِنَ الصَّرْفِ + بِـ + الْإِضَافَةُ] |
| 17 | J'ai vu les deux étudiantes travailleuses chez le grand professeur. | [الْمُثَنَّى + النَّعْتُ + عِنْدَ] |
| 18 | Cette belle voiture rouge est à ton frère. | [اسْمُ الْإِشَارَةِ + النَّعْتُ + لِـ + الْأَسْمَاءُ الْخَمْسَةُ] |
| 19 | Les étudiants qui ont réussi sont allés avec leur professeur avant l'examen. | [الِاسْمُ الْمَوْصُولُ + مَعَ + قَبْلَ] |
| 20 | L'étudiant a écrit une longue lettre pour sa mère avec le stylo. | [النَّعْتُ + لِـ + ضَمِير + بِـ] |
| 21 | Combien de livres as-tu chez ton père ? | [اسْمُ الِاسْتِفْهَامِ + عِنْدَ + الْأَسْمَاءُ الْخَمْسَةُ] |
| 22 | Fatima n'a pas la voiture rouge de son frère. | [مَمْنُوعٌ مِنَ الصَّرْفِ + عِنْدَ نَفْي + النَّعْتُ + الْأَسْمَاءُ الْخَمْسَةُ] |
| 23 | Les deux garçons ont voyagé de la ville au désert en train. | [الْمُثَنَّى + حَرْفُ جَرٍّ + مَمْنُوعٌ مِنَ الصَّرْفِ + بِـ] |
| 24 | Étudie (m.) la leçon que le professeur a expliquée avant l'examen ! | [الْأَمْرُ + الِاسْمُ الْمَوْصُولُ + قَبْلَ] |
| 25 | À qui est ce nouveau stylo sur la table ? | [لِـ + اسْمُ الْإِشَارَةِ + النَّعْتُ + حَرْفُ جَرٍّ] |
| 26 | Son père (à elle) est passé par les grandes mosquées. | [الْأَسْمَاءُ الْخَمْسَةُ + مَرَّ بِـ + مَمْنُوعٌ مِنَ الصَّرْفِ + النَّعْتُ] |
| 27 | Je n'ai pas de frère mais j'ai trois sœurs. | [عِنْدَ نَفْي + الْعَدَدُ] |
| 28 | Ne va (m.) pas chez Ahmed avant le professeur ! | [النَّهْيُ + عِنْدَ + مَمْنُوعٌ مِنَ الصَّرْفِ + قَبْلَ] |
| 29 | Les étudiants travailleurs ont écrit les devoirs ensemble. | [النَّعْتُ + الْمَاضِي + مَعًا] |
| 30 | Quelle est la voiture de ton père ? | [اسْمُ الِاسْتِفْهَامِ + الْإِضَافَةُ + الْأَسْمَاءُ الْخَمْسَةُ] |

> [!tip]- Correction
> 16. ذَهَبَ **أَبُو** أَحْمَدَ إِلَى **مَسَاجِدِ** الْمَدِينَةِ **بِالسَّيَّارَةِ**.
> (**أَبُو** = مَرْفُوعٌ بِالْوَاوِ / **أَحْمَدَ** = مَمْنُوعٌ → مَجْرُورٌ بِالْفَتْحَةِ / **مَسَاجِدِ** = مَمْنُوعٌ لَكِنْ مُضَافٌ → كَسْرَةٌ تَرْجِعُ)
>
> 17. رَأَيْتُ **الطَّالِبَتَيْنِ الْمُجْتَهِدَتَيْنِ** عِنْدَ الْمُعَلِّمِ الْكَبِيرِ.
> (**الطَّالِبَتَيْنِ** = مَنْصُوبٌ بِالْيَاءِ مُثَنَّى / **الْمُجْتَهِدَتَيْنِ** = نَعْتٌ مُثَنَّى مَنْصُوبٌ بِالْيَاءِ / **الْكَبِيرِ** = نَعْتٌ مَجْرُورٌ)
>
> 18. هَذِهِ السَّيَّارَةُ الْجَمِيلَةُ الْحَمْرَاءُ لِأَخِيكَ.
> (**هَذِهِ** = مُؤَنَّثٌ / **الْحَمْرَاءُ** = نَعْتٌ ثَانٍ / **لِأَخِيكَ** = مَجْرُورٌ بِالْيَاءِ)
>
> 19. ذَهَبَ الطُّلَّابُ **الَّذِينَ** نَجَحُوا **مَعَ** مُعَلِّمِهِمْ **قَبْلَ** الِامْتِحَانِ.
> (**الَّذِينَ نَجَحُوا** = صِلَةُ الْمَوْصُولِ / **مُعَلِّمِهِمْ** = مُضَافٌ إِلَيْهِ)
>
> 20. كَتَبَ الطَّالِبُ رِسَالَةً **طَوِيلَةً** **لِأُمِّهِ** **بِالْقَلَمِ**.
> (**طَوِيلَةً** = نَعْتٌ مَنْصُوبٌ مُؤَنَّثٌ / **لِأُمِّهِ** = لِـ + مَجْرُورٌ + ضَمِيرٌ / **بِالْقَلَمِ** = أَدَاةٌ)
>
> 21. كَمْ كِتَابًا عِنْدَكَ عِنْدَ **أَبِيكَ** ؟
> (**كَمْ** = اسْمُ اسْتِفْهَامٍ + تَمْيِيزٌ مَنْصُوبٌ / **أَبِيكَ** = مَجْرُورٌ بِالْيَاءِ)
>
> 22. لَيْسَ عِنْدَ **فَاطِمَةَ** السَّيَّارَةُ الْحَمْرَاءُ لِ**أَخِيهَا**.
> (**فَاطِمَةَ** = مَمْنُوعٌ → فَتْحَةٌ / **الْحَمْرَاءُ** = نَعْتٌ / **أَخِيهَا** = مَجْرُورٌ بِالْيَاءِ)
>
> 23. سَافَرَ **الْوَلَدَانِ** مِنَ الْمَدِينَةِ إِلَى **صَحْرَاءَ** بِالْقِطَارِ.
> (**الْوَلَدَانِ** = فَاعِلٌ مَرْفُوعٌ بِالْأَلِفِ / **صَحْرَاءَ** = مَمْنُوعٌ → فَتْحَةٌ)
>
> 24. **اُدْرُسِ** الدَّرْسَ **الَّذِي** شَرَحَهُ الْمُعَلِّمُ **قَبْلَ** الِامْتِحَانِ !
> (**اُدْرُسْ** = فِعْلُ أَمْرٍ / **الَّذِي شَرَحَهُ** = صِلَةٌ + عَائِدٌ / **قَبْلَ** = ظَرْفٌ)
>
> 25. **لِمَنْ** هَذَا الْقَلَمُ **الْجَدِيدُ** عَلَى الطَّاوِلَةِ ؟
> (**لِمَنْ** = لِـ + مَنْ / **الْجَدِيدُ** = نَعْتٌ مَرْفُوعٌ / **عَلَى** = حَرْفُ جَرٍّ)
>
> 26. مَرَّ **أَبُوهَا** **بِمَسَاجِدَ** كَبِيرَةٍ.
> (**أَبُوهَا** = مَرْفُوعٌ بِالْوَاوِ / **بِمَسَاجِدَ** = مَمْنُوعٌ → فَتْحَةٌ بَعْدَ بِـ / **كَبِيرَةٍ** = نَعْتٌ مَجْرُورٌ نَكِرَةٌ)
>
> 27. مَا عِنْدِي أَخٌ وَلَكِنْ عِنْدِي **ثَلَاثُ** أَخَوَاتٍ.
> (**ثَلَاثُ** بِدُونِ تَاءٍ = أُخْتٌ مُؤَنَّثٌ → الْعَدَدُ يُخَالِفُ)
>
> 28. **لَا تَذْهَبْ** عِنْدَ **أَحْمَدَ** **قَبْلَ** الْمُعَلِّمِ !
> (**لَا تَذْهَبْ** = نَهْيٌ مَجْزُومٌ / **أَحْمَدَ** = مَمْنُوعٌ → فَتْحَةٌ)
>
> 29. كَتَبَ الطُّلَّابُ **الْمُجْتَهِدُونَ** الْوَاجِبَاتِ **مَعًا**.
> (**الْمُجْتَهِدُونَ** = نَعْتٌ جَمْعُ مُذَكَّرٍ سَالِمٍ مَرْفُوعٌ بِالْوَاوِ / **مَعًا** = تَنْوِينُ فَتْحٍ)
>
> 30. **أَيُّ** سَيَّارَةٍ لِ**أَبِيكَ** ؟
> (**أَيُّ** = اسْمُ اسْتِفْهَامٍ مُضَافٌ / **أَبِيكَ** = مَجْرُورٌ بِالْيَاءِ)

> [!tip]- Correction
> 31. اشْتَرَى **أَبُو** فَاطِمَةَ سَيَّارَةً جَمِيلَةً **حَمْرَاءَ** **لِابْنِهِ** **قَبْلَ** الِامْتِحَانِ.
> (**أَبُو** = مَرْفُوعٌ بِالْوَاوِ / **فَاطِمَةَ** = مَمْنُوعٌ → فَتْحَةٌ / **حَمْرَاءَ** = مَمْنُوعٌ → فَتْحَةٌ / **لِابْنِهِ** = لِـ + مَجْرُورٌ)
>
> 32. ذَهَبَ **الطَّالِبَانِ الْمُجْتَهِدَانِ اللَّذَانِ** نَجَحَا عِنْدَ **أَحْمَدَ** مَعَ مُعَلِّمِهِمَا.
> (**الطَّالِبَانِ** = مَرْفُوعٌ بِالْأَلِفِ / **الْمُجْتَهِدَانِ** = نَعْتٌ مُثَنَّى / **اللَّذَانِ** = مَوْصُولٌ مُثَنَّى / **أَحْمَدَ** = مَمْنُوعٌ → فَتْحَةٌ)
>
> 33. **لَا تَشْتَرِ** هَذِهِ السَّيَّارَةَ **الْحَمْرَاءَ** **لِأَخِيكَ** **بِمَالِ** **أَبِيكَ** !
> (**لَا تَشْتَرِ** = نَهْيٌ مَجْزُومٌ بِحَذْفِ حَرْفِ الْعِلَّةِ / **لِأَخِيكَ** = مَجْرُورٌ بِالْيَاءِ / **أَبِيكَ** = مَجْرُورٌ بِالْيَاءِ)
>
> 34. كَمْ **مَسْجِدًا** رَأَيْتَ فِي **مُدُنِ** **الصَّحْرَاءِ** ؟
> (**مَسْجِدًا** = تَمْيِيزٌ مَنْصُوبٌ / **مُدُنِ** = مُضَافٌ / **الصَّحْرَاءِ** = مَعَ الـ → الْكَسْرَةُ تَرْجِعُ)
>
> 35. سَلَّمْتُ عَلَى **أَبِي** إِبْرَاهِيمَ **الَّذِي** جَاءَ عِنْدَنَا **بِالطَّائِرَةِ** مَعَ **ابْنَيْهِ**.
> (**أَبِي** = مَجْرُورٌ بِالْيَاءِ / **إِبْرَاهِيمَ** = مَمْنُوعٌ → فَتْحَةٌ / **ابْنَيْهِ** = مُثَنَّى مَجْرُورٌ بِالْيَاءِ)
>
> 36. هَذِهِ الطَّالِبَةُ هِيَ **الَّتِي** عَلَّمَتْنِي الْعَرَبِيَّةَ **قَبْلَ** الْمُعَلِّمِ عِنْدَ **أَبِيهَا**.
> (**عَلَّمَتْنِي** = فِعْلٌ + نُونُ الْوِقَايَةِ + يَاءُ الْمُتَكَلِّمِ / **أَبِيهَا** = مَجْرُورٌ بِالْيَاءِ)
>
> 37. الرِّجَالُ **الطِّوَالُ الَّذِينَ** سَافَرُوا **بِالطَّائِرَةِ** وَصَلُوا **بَعْدَنَا** إِلَى **مَسَاجِدِ** الْمَدِينَةِ الْكَبِيرَةِ.
> (**الطِّوَالُ** = جَمْعُ تَكْسِيرٍ / **مَسَاجِدِ** = مَمْنُوعٌ لَكِنْ مُضَافٌ → كَسْرَةٌ)
>
> 38. لَيْسَ عِنْدَ **أَخِي** فَاطِمَةَ الْكِتَابُ الْأَحْمَرُ **الَّذِي** اشْتَرَيْتُهُ **لِأَبِيهَا**.
> (**أَخِي** = مَجْرُورٌ بِالْيَاءِ / **فَاطِمَةَ** = مَمْنُوعٌ → فَتْحَةٌ / **لِأَبِيهَا** = لِـ + أَبِي مَجْرُورٌ بِالْيَاءِ)
>
> 39. **اُكْتُبِ الدَّرْسَيْنِ اللَّذَيْنِ** شَرَحَهُمَا الْمُعَلِّمُ **بِالْقَلَمِ الْأَحْمَرِ** **قَبْلَ** الِامْتِحَانِ !
> (**الدَّرْسَيْنِ** = مُثَنَّى مَنْصُوبٌ بِالْيَاءِ / **اللَّذَيْنِ** = مَوْصُولٌ مُثَنَّى / **شَرَحَهُمَا** = الْعَائِدُ ضَمِيرُ الْمُثَنَّى)
>
> 40. **لِمَنْ** **هَاتَانِ** السَّيَّارَتَانِ الْجَمِيلَتَانِ **الْحَمْرَاوَانِ** أَمَامَ **مَسَاجِدِ** الْمَدِينَةِ ؟
> (**هَاتَانِ** = إِشَارَةُ مُثَنَّى مُؤَنَّثٍ / **الْحَمْرَاوَانِ** = مُثَنَّى حَمْرَاءَ / **مَسَاجِدِ** = مُضَافٌ → كَسْرَةٌ)
>
> 41. نَجَحَ ابْنُ **أَخِيكَ** **بِاجْتِهَادِهِ** **قَبْلَ** الطُّلَّابِ الْمُجْتَهِدِينَ.
> (**أَخِيكَ** = مَجْرُورٌ بِالْيَاءِ / **بِاجْتِهَادِهِ** = بِـ سَبَبٌ / **الْمُجْتَهِدِينَ** = نَعْتٌ جَمْعُ مُذَكَّرٍ سَالِمٍ مَجْرُورٌ بِالْيَاءِ)
>
> 42. رَأَيْتُ **الطَّالِبَتَيْنِ اللَّتَيْنِ** خَرَجَتَا مِنْ عِنْدِ **أَحْمَدَ** مَعَ الْمُعَلِّمَةِ.
> (**الطَّالِبَتَيْنِ** = مَنْصُوبٌ بِالْيَاءِ / **اللَّتَيْنِ** = مَوْصُولٌ مُثَنَّى مُؤَنَّثٌ / **أَحْمَدَ** = مَمْنُوعٌ → فَتْحَةٌ)
>
> 43. **لِمَاذَا** ذَهَبَ **أَبُوكَ** إِلَى الْمَسَاجِدِ مَعَ **إِبْرَاهِيمَ** **قَبْلَ** الدَّرْسِ ؟
> (**أَبُوكَ** = مَرْفُوعٌ بِالْوَاوِ / **إِبْرَاهِيمَ** = مَمْنُوعٌ → فَتْحَةٌ)
>
> 44. **لَا تَذْهَبُوا** عِنْدَ **أَبِي** **أَحْمَدَ** بِدُونِ **أَخِيهِ** **قَبْلَ** الِامْتِحَانِ !
> (**لَا تَذْهَبُوا** = نَهْيٌ جَمْعٌ مَجْزُومٌ بِحَذْفِ النُّونِ / **أَبِي** = مَجْرُورٌ بِالْيَاءِ / **أَحْمَدَ** = مَمْنُوعٌ → فَتْحَةٌ / **أَخِيهِ** = مَجْرُورٌ بِالْيَاءِ)
>
> 45. هَذِهِ الْكُتُبُ الثَّلَاثَةُ الْكَبِيرَةُ الْحَمْرَاءُ لِأُمِّ الْمُعَلِّمِ **الَّذِي** عَلَّمَنَا.
> (**الْحَمْرَاءُ** = مَمْنُوعٌ مَرْفُوعٌ بِالضَّمَّةِ / **لِأُمِّ** = لِـ + مُضَافٌ / **عَلَّمَنَا** = ضَمِيرُ جَمْعِ الْمُتَكَلِّمِ)
>
> 46. كَتَبَ الطُّلَّابُ **الْمُجْتَهِدُونَ** الدَّرْسَ **بِقَلَمِ** الْمُعَلِّمِ **مَعًا** **قَبْلَ** الِامْتِحَانِ.
> (**الْمُجْتَهِدُونَ** = نَعْتٌ مَرْفُوعٌ بِالْوَاوِ / **بِقَلَمِ** = بِـ + مُضَافٌ)
>
> 47. سَلَّمْتُ عَلَى **بِنْتَيْ أَخِيكَ اللَّتَيْنِ** جَاءَتَا عِنْدَنَا.
> (**بِنْتَيْ** = مُثَنَّى مَجْرُورٌ بِالْيَاءِ وَهُوَ مُضَافٌ / **أَخِيكَ** = مَجْرُورٌ بِالْيَاءِ / **اللَّتَيْنِ** = مَوْصُولٌ مُثَنَّى)
>
> 48. هَلْ عِنْدَ **أَبِي** فَاطِمَةَ مِفْتَاحُ الْمَسْجِدِ الْكَبِيرِ فِي الْمَدِينَةِ ؟
> (**أَبِي** = مَجْرُورٌ بِالْيَاءِ لِأَنَّهُ مُضَافٌ إِلَيْهِ عِنْدَ / **فَاطِمَةَ** = مَمْنُوعٌ → فَتْحَةٌ / **الْكَبِيرِ** = نَعْتٌ مَجْرُورٌ)
>
> 49. الطَّالِبُ **الَّذِي** كَتَبَ الْكِتَابَ **بِالْقَلَمِ الْأَحْمَرِ** ابْنُ **أَبِيكَ**.
> (**الْأَحْمَرِ** = نَعْتٌ مَجْرُورٌ مَمْنُوعٌ مِنَ الصَّرْفِ لَكِنْ مَعَ الـ → كَسْرَةٌ / **أَبِيكَ** = مَجْرُورٌ بِالْيَاءِ)
>
> 50. **لَا تُسَلِّمْ** عَلَى **أَحْمَدَ** وَ**إِبْرَاهِيمَ** فِي الْمَسَاجِدِ **قَبْلَ** الدَّرْسِ مَعَ الطُّلَّابِ !
> (**لَا تُسَلِّمْ** = نَهْيٌ مَجْزُومٌ / **أَحْمَدَ** وَ**إِبْرَاهِيمَ** = مَمْنُوعَانِ → فَتْحَةٌ)

| 60 | Les étudiants travailleurs qui ont voyagé en avion avec le fils du père d'Ibrahim sont arrivés après nous aux mosquées de la belle ville avant le grand examen. | [النَّعْتُ + الِاسْمُ الْمَوْصُولُ + بِـ + مَعَ + الْأَسْمَاءُ الْخَمْسَةُ + مَمْنُوعٌ + بَعْدَ + الْإِضَافَةُ + قَبْلَ] |

> [!tip]- Correction
> 51. جَاءَتِ **الطَّالِبَتَانِ الْمُجْتَهِدَتَانِ اللَّتَانِ** جَاءَتَا عِنْدَ **أَبِي أَحْمَدَ بِالسَّيَّارَةِ** وَعِنْدَهُمَا **ثَلَاثَةُ** كُتُبٍ **حَمْرَاءَ**.
> (**الطَّالِبَتَانِ** = مُثَنَّى مَرْفُوعٌ بِالْأَلِفِ / **الْمُجْتَهِدَتَانِ** = نَعْتٌ مُثَنَّى / **اللَّتَانِ** = مَوْصُولٌ مُثَنَّى مُؤَنَّثٌ / **أَبِي** = مَجْرُورٌ بِالْيَاءِ / **أَحْمَدَ** = مَمْنُوعٌ → فَتْحَةٌ / **ثَلَاثَةُ** = بِالتَّاءِ لِأَنَّ كِتَاب مُذَكَّرٌ → يُخَالِفُ / **حَمْرَاءَ** = مَمْنُوعٌ → فَتْحَةٌ)
>
> 52. **لَا تَشْتَرِ السَّيَّارَتَيْنِ الْحَمْرَاوَيْنِ اللَّتَيْنِ** رَآهُمَا **أَخُوكَ** عِنْدَ **إِبْرَاهِيمَ قَبْلَ** الِامْتِحَانِ !
> (**لَا تَشْتَرِ** = نَهْيٌ مَجْزُومٌ بِحَذْفِ الْعِلَّةِ / **السَّيَّارَتَيْنِ** = مُثَنَّى مَنْصُوبٌ بِالْيَاءِ / **الْحَمْرَاوَيْنِ** = مُثَنَّى حَمْرَاءَ / **اللَّتَيْنِ** = مَوْصُولٌ مُثَنَّى / **أَخُوكَ** = مَرْفُوعٌ بِالْوَاوِ / **إِبْرَاهِيمَ** = مَمْنُوعٌ → فَتْحَةٌ)
>
> 53. سَلَّمَ **أَبُو** فَاطِمَةَ عَلَى **أَخِي** إِبْرَاهِيمَ **الَّذِي** جَاءَ **مَعَ ابْنَيْهِ** إِلَى **مَسَاجِدِ** الْمَدِينَةِ الْكَبِيرَةِ.
> (**أَبُو** = مَرْفُوعٌ بِالْوَاوِ / **فَاطِمَةَ** = مَمْنُوعٌ → فَتْحَةٌ / **أَخِي** = مَجْرُورٌ بِالْيَاءِ / **إِبْرَاهِيمَ** = مَمْنُوعٌ → فَتْحَةٌ / **ابْنَيْهِ** = مُثَنَّى مَجْرُورٌ بِالْيَاءِ / **مَسَاجِدِ** = مُضَافٌ → كَسْرَةٌ تَرْجِعُ)
>
> 54. **كَمْ** كِتَابًا **أَحْمَرَ** اشْتَرَيْتَ **لِأُمِّ أَخِيكَ بِمَالِ أَبِيكَ قَبْلَ** الِامْتِحَانِ ؟
> (**كَمْ** + تَمْيِيزٌ مَنْصُوبٌ / **أَحْمَرَ** = مَمْنُوعٌ → فَتْحَةٌ نَعْتٌ / **أَخِيكَ** = مَجْرُورٌ بِالْيَاءِ / **أَبِيكَ** = مَجْرُورٌ بِالْيَاءِ)
>
> 55. الطَّالِبَةُ **الَّتِي** عَلَّمَتْنِي الْعَرَبِيَّةَ عِنْدَ **أَبِيهَا** ذَهَبَتْ **مَعَ أَخَوَيْهَا** إِلَى **مَسَاجِدِ** الْمَدِينَةِ **بِالطَّائِرَةِ**.
> (**عَلَّمَتْنِي** = نُونُ الْوِقَايَةِ + يَاءُ الْمُتَكَلِّمِ / **أَبِيهَا** = مَجْرُورٌ بِالْيَاءِ / **أَخَوَيْهَا** = مُثَنَّى أَخ مَجْرُورٌ بِالْيَاءِ / **مَسَاجِدِ** = مُضَافٌ → كَسْرَةٌ)
>
> 56. **هَلْ** عِنْدَ **ابْنَيْ أَخِيكَ** مِفْتَاحُ الْمَسْجِدِ الْكَبِيرِ **الْأَحْمَرِ** فِي الْمَدِينَةِ **الَّذِي** بَنَاهُ الْمُعَلِّمُ ؟
> (**ابْنَيْ** = مُثَنَّى مَجْرُورٌ بِالْيَاءِ مُضَافٌ / **أَخِيكَ** = مَجْرُورٌ بِالْيَاءِ / **الْأَحْمَرِ** = مَمْنُوعٌ لَكِنْ مَعَ الـ → كَسْرَةٌ / **بَنَاهُ** = الْعَائِدُ)
>
> 57. رَأَيْتُ **أَبَا** أَحْمَدَ وَ**أَخَا** إِبْرَاهِيمَ **الَّذِينَ** خَرَجُوا **مَعًا** مِنَ الْمَسَاجِدِ الْكَبِيرَةِ **قَبْلَ** الدَّرْسِ.
> (**أَبَا** = مَنْصُوبٌ بِالْأَلِفِ مَفْعُولٌ بِهِ / **أَخَا** = مَنْصُوبٌ بِالْأَلِفِ / **أَحْمَدَ** وَ**إِبْرَاهِيمَ** = مَمْنُوعَانِ → فَتْحَةٌ / **الَّذِينَ** = مَوْصُولٌ جَمْعُ مُذَكَّرٍ)
>
> 58. **لِمَاذَا** لَمْ يَشْتَرِ **أَبُوكَ** هَذِهِ السَّيَّارَةَ الْجَمِيلَةَ **الْحَمْرَاءَ لِأُخْتِكَ قَبْلَ** الِامْتِحَانِ **بِمَالِ أَخِيكَ** ؟
> (**أَبُوكَ** = مَرْفُوعٌ بِالْوَاوِ / **الْحَمْرَاءَ** = مَمْنُوعٌ مَنْصُوبٌ بِالْفَتْحَةِ / **أَخِيكَ** = مَجْرُورٌ بِالْيَاءِ / **لَمْ يَشْتَرِ** = مُضَارِعٌ مَجْزُومٌ بِحَذْفِ الْعِلَّةِ)
>
> 59. **اُكْتُبِ** الدُّرُوسَ الثَّلَاثَةَ **الَّتِي** شَرَحَهَا **الْمُعَلِّمَانِ بِالْقَلَمِ الْأَحْمَرِ** عِنْدَ **أَبِي** فَاطِمَةَ **قَبْلَ** الِامْتِحَانِ !
> (**اُكْتُبْ** = أَمْرٌ / **الَّتِي** = مَوْصُولٌ لِأَنَّ دُرُوس جَمْعُ تَكْسِيرٍ = مُؤَنَّثٌ / **الْمُعَلِّمَانِ** = مُثَنَّى مَرْفُوعٌ بِالْأَلِفِ / **الْأَحْمَرِ** = مَمْنُوعٌ مَعَ الـ → كَسْرَةٌ / **أَبِي** = مَجْرُورٌ بِالْيَاءِ / **فَاطِمَةَ** = مَمْنُوعٌ → فَتْحَةٌ)
>
> 60. وَصَلَ الطُّلَّابُ **الْمُجْتَهِدُونَ الَّذِينَ** سَافَرُوا **بِالطَّائِرَةِ مَعَ** ابْنِ **أَبِي** إِبْرَاهِيمَ **بَعْدَنَا** إِلَى **مَسَاجِدِ** الْمَدِينَةِ الْجَمِيلَةِ **قَبْلَ** الِامْتِحَانِ الْكَبِيرِ.
> (**الْمُجْتَهِدُونَ** = نَعْتٌ مَرْفُوعٌ بِالْوَاوِ / **الَّذِينَ** = مَوْصُولٌ / **أَبِي** = مَجْرُورٌ بِالْيَاءِ / **إِبْرَاهِيمَ** = مَمْنُوعٌ → فَتْحَةٌ / **مَسَاجِدِ** = مُضَافٌ → كَسْرَةٌ تَرْجِعُ)
