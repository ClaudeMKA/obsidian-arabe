## Exercice 1 — Complète avec la bonne lettre (و / ا / ي)

| # | Phrase | Lettre | État |
|---|---|---|---|
| 1 | جاءَ أب...كَ | | مرفوع |
| 2 | رأيتُ أب...كَ | | منصوب |
| 3 | سلّمتُ على أب...كَ | | مجرور |
| 4 | هذا أخ...هُ | | مرفوع |
| 5 | إنَّ أخ...كَ ذكيٌّ | | منصوب |
| 6 | هو ذ... مالٍ | | مرفوع |

> [!tip]- Correction
> 1. أب**و**كَ — مرفوع بالواو
> 2. أب**ا**كَ — منصوب بالألف
> 3. أب**ي**كَ — مجرور بالياء
> 4. أخ**و**هُ — مرفوع بالواو
> 5. أخ**ا**كَ — منصوب بالألف
> 6. ذ**و** مالٍ — مرفوع بالواو

## Exercice 2 — Le piège de ياء المتكلم

Quelle est la déclinaison correcte quand le nom est ajouté à **ي** (mon) ?

| #   | Phrase     | Correct ? |
| --- | ---------- | --------- |
| 1   | جاءَ أبوي  |           |
| 2   | جاءَ أبي   |           |
| 3   | رأيتُ أباي |           |
| 4   | رأيتُ أبي  |           |

> [!tip]- Correction
> 1. **FAUX** — avec ياء المتكلم, les signes spéciaux (و/ا/ي) disparaissent !
> 2. **CORRECT** — جاءَ **أبي** (la déclinaison redevient normale)
> 3. **FAUX** — même raison
> 4. **CORRECT** — رأيتُ **أبي**
>
> Quand les 5 noms sont مضاف à **ي** (mon), ils perdent les signes spéciaux et redeviennent normaux.

## Exercice 3 — Mets le bon إعراب

Complète la phrase et donne le إعراب :

| # | Phrase | Le mot | إعراب |
|---|---|---|---|
| 1 | جاءَ أب... بكرٍ | | |
| 2 | رأيتُ ذ... العلمِ | | |
| 3 | مررتُ بأب... الطالبِ | | |
| 4 | أخ... المعلمِ طبيبٌ | | |
| 5 | إنَّ أب... كَ كريمٌ | | |

> [!tip]- Correction
> 1. أب**و** بكرٍ — مرفوع بالواو (فاعل)
> 2. ذ**ا** العلمِ — منصوب بالألف (مفعول به)
> 3. بأب**ي** الطالبِ — مجرور بالياء (après حرف جر)
> 4. أخ**و** المعلمِ — مرفوع بالواو (مبتدأ)
> 5. أب**ا**كَ — منصوب بالألف (اسم إنَّ)

## Exercice 4 — Rappel : quels sont les 5 noms ?

Cite-les de mémoire :

> [!tip]- Correction
> 1. **أب** — père
> 2. **أخ** — frère
> 3. **حم** — beau-père
> 4. **فو** — bouche
> 5. **ذو** — possesseur de
>
> Condition : ils doivent être **مضاف** (sauf à ياء المتكلم) pour utiliser و/ا/ي
