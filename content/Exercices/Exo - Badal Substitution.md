# Exercices — البدل La Substitution

Cours : [[Badal - La substitution]]

---

## Exercice 1 — Trouve le بدل

Identifie le **بدل** dans chaque phrase :

| # | Phrase | Le بدل | Son إعراب |
|---|---|---|---|
| 1 | هذا **الكتابُ** جميلٌ | | |
| 2 | قرأتُ هذا **الكتابَ** | | |
| 3 | كتبتُ في هذا **الدفترِ** | | |
| 4 | هذه **السيارةُ** جديدةٌ | | |
| 5 | رأيتُ ذلك **الرجلَ** | | |

> [!tip]- Correction
> 1. **الكتابُ** — بدل مرفوع بالضمة (هذا = مبتدأ مرفوع)
> 2. **الكتابَ** — بدل منصوب بالفتحة (هذا = مفعول به منصوب)
> 3. **الدفترِ** — بدل مجرور بالكسرة (هذا = مجرور بـ في)
> 4. **السيارةُ** — بدل مرفوع بالضمة
> 5. **الرجلَ** — بدل منصوب بالفتحة

## Exercice 2 — Le test de suppression

Supprime اسم الإشارة. La phrase reste-t-elle correcte ? Si oui → c'est un بدل.

| # | Phrase complète | Sans اسم الإشارة | بدل ? |
|---|---|---|---|
| 1 | هذا الولدُ ذكيٌّ | الولدُ ذكيٌّ | |
| 2 | هذا كتابٌ | كتابٌ | |
| 3 | تلك المدرسةُ كبيرةٌ | المدرسةُ كبيرةٌ | |

> [!tip]- Correction
> 1. ✅ **Oui** → الولدُ est بدل (la phrase marche sans هذا)
> 2. ❌ **Non** → كتابٌ est **خبر** (pas de ال = pas un بدل, c'est une phrase nominale "c'est un livre")
> 3. ✅ **Oui** → المدرسةُ est بدل

## Exercice 3 — Donne le إعراب complet

Fais l'analyse grammaticale complète du بدل :

| # | Phrase | إعراب du بدل |
|---|---|---|
| 1 | هذا المعلمُ ممتازٌ | |
| 2 | سلّمتُ على هذه الطالبةِ | |
| 3 | إنَّ هذا الكتابَ مفيدٌ | |

> [!tip]- Correction
> 1. **المعلمُ** : بدل مرفوع وعلامة رفعه **الضمة** (car هذا = مبتدأ مرفوع)
> 2. **الطالبةِ** : بدل مجرور وعلامة جره **الكسرة** (car هذه = مجرور بعلى)
> 3. **الكتابَ** : بدل منصوب وعلامة نصبه **الفتحة** (car هذا = اسم إنَّ منصوب)
