## Exercice 2 — Quel est le إعراب du mot après le حرف جر ?

| # | Phrase | إعراب du mot en gras |
|---|---|---|
| 1 | ذهبتُ إلى **المدرسةِ** | |
| 2 | الكتابُ على **الطاولةِ** | |
| 3 | خرجَ من **البيتِ** | |
| 4 | سلّمتُ على **أحمدَ** | |
| 5 | صلّيتُ في **المساجدِ** | |

> [!tip]- Correction
> Tous sont **مجرور** ! C'est la règle : tout اسم après un حرف جر est مجرور.
> 1. المدرسةِ — مجرور بالكسرة
> 2. الطاولةِ — مجرور بالكسرة
> 3. البيتِ — مجرور بالكسرة
> 4. أحمدَ — مجرور بال**فتحة** (ممنوع من الصرف !)
> 5. المساجدِ — مجرور بالكسرة (مع ال → كسرة)

## Exercice 4 — Piège : بـ / لـ / كـ se collent au mot !

Écris correctement :

| # | حرف جر + mot | Forme correcte |
|---|---|---|
| 1 | بـ + القلم | |
| 2 | لـ + الطالب | |
| 3 | كـ + الأسد | |
| 4 | بـ + سيارة | |

> [!tip]- Correction
> 1. **بالقلمِ** (بـ + ال → بال)
> 2. **للطالبِ** (لـ + ال → لِلْ)
> 3. **كالأسدِ** (كـ + ال → كال)
> 4. **بسيارةٍ** (بـ + سيارة sans ال)
