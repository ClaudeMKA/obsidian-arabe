# Exercices — اسم الاستفهام Les Interrogatifs

Cours : [[Ism Istifham - Interrogatifs]]

---

## Exercice 1 — Quel mot interrogatif ?

Complète avec **مَن / ما / ماذا / متى / أين / كيف / كم / لماذا / هل** :

| # | Question | Mot interrogatif |
|---|---|---|
| 1 | ... هذا الرجلُ ؟ (Qui est cet homme ?) | |
| 2 | ... اسمُكَ ؟ (Quel est ton nom ?) | |
| 3 | ... الامتحانُ ؟ (Quand est l'examen ?) | |
| 4 | ... المسجدُ ؟ (Où est la mosquée ?) | |
| 5 | ... حالُكَ ؟ (Comment vas-tu ?) | |
| 6 | ... كتابًا عندَكَ ؟ (Combien de livres as-tu ?) | |
| 7 | ... تدرسُ ؟ (Pourquoi tu étudies ?) | |
| 8 | ... أنتَ طالبٌ ؟ (Es-tu étudiant ?) | |
| 9 | ... تأكلُ ؟ (Que manges-tu ?) | |
| 10 | ... جاءَ ؟ (Qui est venu ?) | |

> [!tip]- Correction
> 1. **مَن** (qui — pour les personnes عاقل)
> 2. **ما** (quoi/quel — pour les choses)
> 3. **متى** (quand — le temps)
> 4. **أينَ** (où — le lieu)
> 5. **كيفَ** (comment — la manière)
> 6. **كم** (combien — le nombre)
> 7. **لماذا** (pourquoi — la cause)
> 8. **هل** (est-ce que — réponse نعم/لا)
> 9. **ماذا** (que/quoi — pour les choses)
> 10. **مَن** (qui)

## Exercice 2 — مَن ou ما ?

Le piège : **مَن** = personnes, **ما** = choses

| # | Question | مَن ou ما ? |
|---|---|---|
| 1 | ... هذا ؟ (en montrant un homme) | |
| 2 | ... هذا ؟ (en montrant un objet) | |
| 3 | ... في الحقيبةِ ؟ | |
| 4 | ... عندَ البابِ ؟ | |
| 5 | ... لونُ السيارةِ ؟ | |

> [!tip]- Correction
> 1. **مَن** (on demande l'identité d'une personne)
> 2. **ما** (on demande la nature d'une chose)
> 3. **ما** / **ماذا** (qu'y a-t-il dans le sac ?)
> 4. **مَن** (qui est à la porte ?)
> 5. **ما** (quelle est la couleur ?)

## Exercice 3 — Piège : هل est un حرف, pas un اسم !

Vrai ou Faux :

| # | Affirmation | V/F |
|---|---|---|
| 1 | هل est un اسم استفهام | |
| 2 | أ (hamza) est un حرف استفهام | |
| 3 | On peut répondre à مَن par نعم/لا | |
| 4 | On peut répondre à هل par نعم/لا | |
| 5 | أيّ est toujours suivi d'un مضاف إليه | |

> [!tip]- Correction
> 1. **FAUX** — هل est un **حرف** استفهام (pas un اسم)
> 2. **VRAI** — أ est un حرف استفهام
> 3. **FAUX** — مَن attend une réponse avec un nom, pas نعم/لا
> 4. **VRAI** — هل أنتَ طالبٌ ؟ → نعم / لا
> 5. **VRAI** — أيُّ كتابٍ ؟ (quel livre ?)

## Exercice 4 — Pose la question

Écris la question en arabe pour chaque réponse :

| # | Réponse | Question |
|---|---|---|
| 1 | هذا محمدٌ | ... ؟ |
| 2 | اسمي أحمدُ | ... ؟ |
| 3 | نعم، أنا طالبٌ | ... ؟ |
| 4 | المسجدُ هناكَ | ... ؟ |
| 5 | عندي ثلاثةُ كتبٍ | ... ؟ |

> [!tip]- Correction
> 1. **مَن هذا ؟** (Qui est-ce ?)
> 2. **ما اسمُكَ ؟** (Quel est ton nom ?)
> 3. **هل أنتَ طالبٌ ؟** (Es-tu étudiant ?)
> 4. **أينَ المسجدُ ؟** (Où est la mosquée ?)
> 5. **كم كتابًا عندَكَ ؟** (Combien de livres as-tu ?)
