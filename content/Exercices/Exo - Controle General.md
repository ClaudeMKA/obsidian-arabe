## Partie 1 — أَنْوَاعُ الْكَلِمَةِ (Types de mots) `/10`

Classe chaque mot : **اِسْمٌ** (nom), **فِعْلٌ** (verbe) ou **حَرْفٌ** (particule) :

| # | الْكَلِمَةُ | اِسْمٌ / فِعْلٌ / حَرْفٌ |
|---|---|---|
| 1 | كَتَبَ | |
| 2 | فِي | |
| 3 | مُعَلِّمٌ | |
| 4 | يَذْهَبُ | |
| 5 | عَلَى | |
| 6 | مَدْرَسَةٌ | |
| 7 | اُكْتُبْ | |
| 8 | هَلْ | |
| 9 | جَمِيلٌ | |
| 10 | مِنْ | |

> [!tip]- Correction
> 1. كَتَبَ → **فِعْلٌ** (مَاضٍ)
> 2. فِي → **حَرْفٌ** (حَرْفُ جَرٍّ)
> 3. مُعَلِّمٌ → **اِسْمٌ**
> 4. يَذْهَبُ → **فِعْلٌ** (مُضَارِعٌ)
> 5. عَلَى → **حَرْفٌ** (حَرْفُ جَرٍّ)
> 6. مَدْرَسَةٌ → **اِسْمٌ**
> 7. اُكْتُبْ → **فِعْلٌ** (أَمْرٌ)
> 8. هَلْ → **حَرْفٌ** (حَرْفُ اسْتِفْهَامٍ)
> 9. جَمِيلٌ → **اِسْمٌ** (صِفَةٌ = اِسْمٌ en arabe !)
> 10. مِنْ → **حَرْفٌ** (حَرْفُ جَرٍّ)
>
> **Rappel :** En arabe, les adjectifs sont des **أَسْمَاءٌ** (noms).

## Partie 2 — الْمَعْرِفَةُ وَالنَّكِرَةُ (Défini et indéfini) `/10`

Indique si le mot est **مَعْرِفَةٌ** (défini) ou **نَكِرَةٌ** (indéfini) et pourquoi :

| # | الْكَلِمَةُ | مَعْرِفَةٌ أَوْ نَكِرَةٌ ؟ | السَّبَبُ (la raison) |
|---|---|---|---|
| 1 | كِتَابٌ | | |
| 2 | الْكِتَابُ | | |
| 3 | كِتَابُهُ | | |
| 4 | هَذَا | | |
| 5 | مُحَمَّدٌ | | |
| 6 | رَجُلٌ | | |
| 7 | أَنْتَ | | |
| 8 | الَّذِي | | |
| 9 | مُعَلِّمُ الْمَدْرَسَةِ | | |
| 10 | بَيْتٌ كَبِيرٌ | | |

> [!tip]- Correction
> 1. كِتَابٌ → **نَكِرَةٌ** (tanwīn = indéfini)
> 2. الْكِتَابُ → **مَعْرِفَةٌ** (الـ = article défini)
> 3. كِتَابُهُ → **مَعْرِفَةٌ** (مُضَافٌ إِلَى ضَمِيرٍ)
> 4. هَذَا → **مَعْرِفَةٌ** (اِسْمُ إِشَارَةٍ)
> 5. مُحَمَّدٌ → **مَعْرِفَةٌ** (عَلَمٌ = nom propre)
> 6. رَجُلٌ → **نَكِرَةٌ** (tanwīn)
> 7. أَنْتَ → **مَعْرِفَةٌ** (ضَمِيرٌ)
> 8. الَّذِي → **مَعْرِفَةٌ** (اِسْمٌ مَوْصُولٌ)
> 9. مُعَلِّمُ الْمَدْرَسَةِ → **مَعْرِفَةٌ** (مُضَافٌ إِلَى مَعْرِفَةٍ)
> 10. بَيْتٌ كَبِيرٌ → **نَكِرَةٌ** (tanwīn)

## Partie 6 — اِسْمُ الْاِسْتِفْهَامِ (Interrogatifs) `/6`

Choisis le bon interrogatif : مَنْ / مَا / مَاذَا / أَيْنَ / كَيْفَ / كَمْ / مَتَى / لِمَاذَا / هَلْ

| # | Question | Interrogatif |
|---|---|---|
| 1 | ... هَذَا ؟ — هَذَا كِتَابٌ | |
| 2 | ... فَعَلَ هَذَا ؟ — مُحَمَّدٌ فَعَلَ هَذَا | |
| 3 | ... حَالُكَ ؟ — بِخَيْرٍ | |
| 4 | ... تَسْكُنُ ؟ — فِي بَارِيسَ | |
| 5 | ... عِنْدَكَ كِتَابٌ ؟ — نَعَمْ | |
| 6 | ... كِتَابًا عِنْدَكَ ؟ — خَمْسَةٌ | |

> [!tip]- Correction
> 1. **مَا** هَذَا ؟ (pour les choses)
> 2. **مَنْ** فَعَلَ هَذَا ؟ (pour les personnes)
> 3. **كَيْفَ** حَالُكَ ؟ (comment)
> 4. **أَيْنَ** تَسْكُنُ ؟ (où)
> 5. **هَلْ** عِنْدَكَ كِتَابٌ ؟ (oui/non)
> 6. **كَمْ** كِتَابًا عِنْدَكَ ؟ (combien)

## Partie 7 — الْأَسْمَاءُ الْخَمْسَةُ (Les 5 noms) `/6`

Complète avec la bonne forme de أَبٌ (père) :

| # | Phrase | الْإِعْرَابُ |
|---|---|---|
| 1 | جَاءَ ... مُحَمَّدٍ | مَرْفُوعٌ |
| 2 | رَأَيْتُ ... مُحَمَّدٍ | مَنْصُوبٌ |
| 3 | سَلَّمْتُ عَلَى ... مُحَمَّدٍ | مَجْرُورٌ |

Et avec أَخٌ (frère) :

| # | Phrase | الْإِعْرَابُ |
|---|---|---|
| 4 | ... كَ كَرِيمٌ | مَرْفُوعٌ |
| 5 | أُحِبُّ ... كَ | مَنْصُوبٌ |
| 6 | ذَهَبْتُ مَعَ ... كَ | مَجْرُورٌ |

> [!tip]- Correction
> 1. جَاءَ **أَبُو** مُحَمَّدٍ (مَرْفُوعٌ بِالْوَاوِ)
> 2. رَأَيْتُ **أَبَا** مُحَمَّدٍ (مَنْصُوبٌ بِالْأَلِفِ)
> 3. سَلَّمْتُ عَلَى **أَبِي** مُحَمَّدٍ (مَجْرُورٌ بِالْيَاءِ)
> 4. **أَخُو**كَ كَرِيمٌ (مَرْفُوعٌ بِالْوَاوِ)
> 5. أُحِبُّ **أَخَا**كَ (مَنْصُوبٌ بِالْأَلِفِ)
> 6. ذَهَبْتُ مَعَ **أَخِي**كَ (مَجْرُورٌ بِالْيَاءِ)
>
> **Rappel :** Les الْأَسْمَاءُ الْخَمْسَةُ se déclinent avec و (مَرْفُوعٌ), ا (مَنْصُوبٌ), ي (مَجْرُورٌ) **uniquement** quand ils sont مُضَافٌ.

## Partie 8 — النَّعْتُ وَالْمَنْعُوتُ (Adjectif) `/8`

Corrige les erreurs d'accord dans ces phrases :

| # | Phrase (avec erreur) | Correction |
|---|---|---|
| 1 | الطَّالِبُ الْجَمِيلَةُ | |
| 2 | الْمُعَلِّمَةُ الْكَبِيرُ | |
| 3 | كِتَابٌ جَدِيدَةٌ | |
| 4 | طَالِبَانِ مُجْتَهِدٌ | |
| 5 | الرِّجَالُ الطَّوِيلُ | |
| 6 | سَيَّارَةٌ أَحْمَرٌ | |
| 7 | رَأَيْتُ رَجُلًا طَوِيلٌ | |
| 8 | فِي الْمَدْرَسَةِ الْكَبِيرُ | |

> [!tip]- Correction
> 1. الطَّالِبُ **الْجَمِيلُ** (مُذَكَّرٌ → مُذَكَّرٌ)
> 2. الْمُعَلِّمَةُ **الْكَبِيرَةُ** (مُؤَنَّثٌ → مُؤَنَّثٌ)
> 3. كِتَابٌ **جَدِيدٌ** (مُذَكَّرٌ → مُذَكَّرٌ)
> 4. طَالِبَانِ **مُجْتَهِدَانِ** (مُثَنَّى → مُثَنَّى)
> 5. الرِّجَالُ **الطِّوَالُ** (جَمْعٌ → جَمْعُ تَكْسِيرٍ)
> 6. سَيَّارَةٌ **حَمْرَاءُ** (مُؤَنَّثٌ → أَفْعَلُ → فَعْلَاءُ)
> 7. رَأَيْتُ رَجُلًا **طَوِيلًا** (مَنْصُوبٌ → مَنْصُوبٌ ! النَّعْتُ يَتْبَعُ الْمَنْعُوتَ فِي الْإِعْرَابِ)
> 8. فِي الْمَدْرَسَةِ **الْكَبِيرَةِ** (مَجْرُورٌ مُؤَنَّثٌ → مَجْرُورٌ مُؤَنَّثٌ)
>
> **Règle :** النَّعْتُ يَتْبَعُ الْمَنْعُوتَ فِي 4 choses : الْإِعْرَابُ + التَّعْرِيفُ وَالتَّنْكِيرُ + الْمُذَكَّرُ وَالْمُؤَنَّثُ + الْمُفْرَدُ وَالْمُثَنَّى وَالْجَمْعُ

## Partie 9 — الْمَمْنُوعُ مِنَ الصَّرْفِ `/8`

### 9a. Ce mot est-il مَمْنُوعٌ مِنَ الصَّرْفِ ? Pourquoi ?

| # | الْكَلِمَةُ | مَمْنُوعٌ ؟ | السَّبَبُ |
|---|---|---|---|
| 1 | أَحْمَدُ | | |
| 2 | كِتَابٌ | | |
| 3 | فَاطِمَةُ | | |
| 4 | مَسَاجِدُ | | |
| 5 | أَحْمَرُ | | |
| 6 | صَحْرَاءُ | | |

> [!tip]- Correction
> 1. **نَعَمْ** — عَلَمٌ عَلَى وَزْنِ أَفْعَلَ
> 2. **لَا** — اِسْمٌ عَادِيٌّ (كِتَابٌ ← tanwīn possible)
> 3. **نَعَمْ** — عَلَمٌ مُؤَنَّثٌ
> 4. **نَعَمْ** — صِيغَةُ مُنْتَهَى الْجُمُوعِ (مَفَاعِلُ)
> 5. **نَعَمْ** — صِفَةٌ عَلَى وَزْنِ أَفْعَلَ
> 6. **نَعَمْ** — أَلِفُ التَّأْنِيثِ الْمَمْدُودَةُ (اء)

### 9b. Mets la bonne حَرَكَة sur le dernier mot :

| # | Phrase | كَسْرَةٌ ou فَتْحَةٌ ? |
|---|---|---|
| 1 | سَلَّمْتُ عَلَى أَحْمَد... | |
| 2 | ذَهَبْتُ إِلَى مَسَاجِد... كَثِيرَةٍ | |
| 3 | صَلَّيْتُ فِي **الْ**مَسَاجِد... | |
| 4 | مَرَرْتُ بِصَحْرَاء... وَاسِعَةٍ | |

> [!tip]- Correction
> 1. أَحْمَد**َ** → **فَتْحَةٌ** (مَمْنُوعٌ مِنَ الصَّرْفِ)
> 2. مَسَاجِد**َ** → **فَتْحَةٌ** (مَمْنُوعٌ مِنَ الصَّرْفِ, sans الـ ni إِضَافَةٌ)
> 3. **الْ**مَسَاجِد**ِ** → **كَسْرَةٌ** ! (مَعَ الـ → الْكَسْرَةُ تَرْجِعُ)
> 4. صَحْرَاء**َ** → **فَتْحَةٌ** (مَمْنُوعٌ مِنَ الصَّرْفِ)

## Partie 10 — حُرُوفُ الْجَرِّ (Prépositions) `/10`

### 10a. Traduis avec la bonne préposition :

| # | Français | Arabe |
|---|---|---|
| 1 | Je suis allé **à** l'école | |
| 2 | Il est sorti **de** la maison | |
| 3 | Le livre est **sur** la table | |
| 4 | Il est **dans** la classe | |
| 5 | J'ai écrit **avec** le stylo | |

> [!tip]- Correction
> 1. ذَهَبْتُ **إِلَى** الْمَدْرَسَةِ
> 2. خَرَجَ **مِنَ** الْبَيْتِ
> 3. الْكِتَابُ **عَلَى** الطَّاوِلَةِ
> 4. هُوَ **فِي** الْفَصْلِ
> 5. كَتَبْتُ **بِـ**الْقَلَمِ

### 10b. عِنْدَ ou لِـ ?

| # | Phrase | عِنْدَ ou لِـ ? | Arabe |
|---|---|---|---|
| 1 | J'ai un livre (chez moi) | | |
| 2 | Ce livre est à Mohammed | | |
| 3 | Nous avons un examen | | |
| 4 | À qui est cette voiture ? | | |
| 5 | Elle n'a pas de stylo | | |

> [!tip]- Correction
> 1. **عِنْدِي** كِتَابٌ → عِنْدَ (possession physique)
> 2. الْكِتَابُ **لِمُحَمَّدٍ** → لِـ (propriété)
> 3. **عِنْدَنَا** اِمْتِحَانٌ → عِنْدَ
> 4. **لِمَنْ** هَذِهِ السَّيَّارَةُ ؟ → لِـ (demander la propriété)
> 5. **مَا عِنْدَهَا** قَلَمٌ → عِنْدَ (ou لَيْسَ عِنْدَهَا)

## Partie 11 — بِـ وَ مَعَ (Avec) `/6`

Complète avec **بِـ** ou **مَعَ** :

| # | Phrase | بِـ ou مَعَ ? |
|---|---|---|
| 1 | ذَهَبْتُ ... صَدِيقِي | |
| 2 | كَتَبْتُ ... الْقَلَمِ | |
| 3 | سَافَرْتُ ... الطَّائِرَةِ | |
| 4 | جَاءَتْ ... أُخْتِهَا | |
| 5 | قَطَعَ الْخُبْزَ ... السِّكِّينِ | |
| 6 | دَرَسْنَا ... | (ensemble) |

> [!tip]- Correction
> 1. **مَعَ** صَدِيقِي (accompagnement → personne)
> 2. **بِـ**الْقَلَمِ (instrument → objet)
> 3. **بِـ**الطَّائِرَةِ (moyen de transport)
> 4. **مَعَ** أُخْتِهَا (accompagnement → personne)

> 6. دَرَسْنَا **مَعًا** (ensemble → tanwīn fatha)

## Partie 16 — الْجُمْلَةُ الْفِعْلِيَّةُ (Phrase verbale) `/8`

Analyse chaque phrase : identifie le **فِعْلٌ**, le **فَاعِلٌ** et le **مَفْعُولٌ بِهِ** (s'il y en a) :

| # | Phrase | الْفِعْلُ | الْفَاعِلُ | الْمَفْعُولُ بِهِ |
|---|---|---|---|---|
| 1 | كَتَبَ الطَّالِبُ الدَّرْسَ | | | |
| 2 | ذَهَبَ مُحَمَّدٌ | | | |
| 3 | أَكَلَتْ فَاطِمَةُ التُّفَّاحَةَ | | | |
| 4 | يَدْرُسُ الطُّلَّابُ | | | |
| 5 | فَتَحَ الْمُعَلِّمُ الْبَابَ | | | |
| 6 | نَامَ الطِّفْلُ | | | |

> [!tip]- Correction
> 1. فِعْلٌ: **كَتَبَ** / فَاعِلٌ: **الطَّالِبُ** (مَرْفُوعٌ بِالضَّمَّةِ) / مَفْعُولٌ بِهِ: **الدَّرْسَ** (مَنْصُوبٌ بِالْفَتْحَةِ)
> 2. فِعْلٌ: **ذَهَبَ** / فَاعِلٌ: **مُحَمَّدٌ** / مَفْعُولٌ بِهِ: ❌ (فِعْلٌ لَازِمٌ)
> 3. فِعْلٌ: **أَكَلَتْ** / فَاعِلٌ: **فَاطِمَةُ** / مَفْعُولٌ بِهِ: **التُّفَّاحَةَ**
> 4. فِعْلٌ: **يَدْرُسُ** / فَاعِلٌ: **الطُّلَّابُ** / مَفْعُولٌ بِهِ: ❌ (لَازِمٌ)
> 5. فِعْلٌ: **فَتَحَ** / فَاعِلٌ: **الْمُعَلِّمُ** / مَفْعُولٌ بِهِ: **الْبَابَ**
> 6. فِعْلٌ: **نَامَ** / فَاعِلٌ: **الطِّفْلُ** / مَفْعُولٌ بِهِ: ❌ (لَازِمٌ)
>
> **Rappel :** الْفَاعِلُ toujours **مَرْفُوعٌ**. الْمَفْعُولُ بِهِ toujours **مَنْصُوبٌ**.

## Partie 17 — الْإِعْرَابُ الشَّامِلُ (Analyse grammaticale) `/12`

Fais le إِعْرَابٌ complet de ces phrases :

### Phrase 1 : ذَهَبَ الطَّالِبُ إِلَى الْمَدْرَسَةِ

| الْكَلِمَةُ | الْإِعْرَابُ |
|---|---|
| ذَهَبَ | |
| الطَّالِبُ | |
| إِلَى | |
| الْمَدْرَسَةِ | |

> [!tip]- Correction
> - **ذَهَبَ** : فِعْلٌ مَاضٍ مَبْنِيٌّ عَلَى الْفَتْحِ
> - **الطَّالِبُ** : فَاعِلٌ مَرْفُوعٌ وَعَلَامَةُ رَفْعِهِ الضَّمَّةُ
> - **إِلَى** : حَرْفُ جَرٍّ
> - **الْمَدْرَسَةِ** : اِسْمٌ مَجْرُورٌ بِـ(إِلَى) وَعَلَامَةُ جَرِّهِ الْكَسْرَةُ

### Phrase 2 : قَرَأَ أَبُو مُحَمَّدٍ كِتَابًا جَدِيدًا

| الْكَلِمَةُ | الْإِعْرَابُ |
|---|---|
| قَرَأَ | |
| أَبُو | |
| مُحَمَّدٍ | |
| كِتَابًا | |
| جَدِيدًا | |

> [!tip]- Correction
> - **قَرَأَ** : فِعْلٌ مَاضٍ مَبْنِيٌّ عَلَى الْفَتْحِ
> - **أَبُو** : فَاعِلٌ مَرْفُوعٌ وَعَلَامَةُ رَفْعِهِ **الْوَاوُ** (مِنَ الْأَسْمَاءِ الْخَمْسَةِ)
> - **مُحَمَّدٍ** : مُضَافٌ إِلَيْهِ مَجْرُورٌ وَعَلَامَةُ جَرِّهِ الْكَسْرَةُ
> - **كِتَابًا** : مَفْعُولٌ بِهِ مَنْصُوبٌ وَعَلَامَةُ نَصْبِهِ الْفَتْحَةُ
> - **جَدِيدًا** : نَعْتٌ مَنْصُوبٌ وَعَلَامَةُ نَصْبِهِ الْفَتْحَةُ (يَتْبَعُ الْمَنْعُوتَ)

### Phrase 3 : سَلَّمْتُ عَلَى أَحْمَدَ فِي مَسَاجِدِ الْمَدِينَةِ

| الْكَلِمَةُ | الْإِعْرَابُ |
|---|---|
| سَلَّمْتُ | |
| عَلَى | |
| أَحْمَدَ | |
| فِي | |
| مَسَاجِدِ | |
| الْمَدِينَةِ | |

> [!tip]- Correction
> - **سَلَّمْتُ** : فِعْلٌ مَاضٍ مَبْنِيٌّ عَلَى السُّكُونِ، وَالتَّاءُ ضَمِيرٌ مُتَّصِلٌ فِي مَحَلِّ رَفْعِ فَاعِلٍ
> - **عَلَى** : حَرْفُ جَرٍّ
> - **أَحْمَدَ** : اِسْمٌ مَجْرُورٌ وَعَلَامَةُ جَرِّهِ **الْفَتْحَةُ** نِيَابَةً عَنِ الْكَسْرَةِ (مَمْنُوعٌ مِنَ الصَّرْفِ)
> - **فِي** : حَرْفُ جَرٍّ
> - **مَسَاجِدِ** : اِسْمٌ مَجْرُورٌ وَعَلَامَةُ جَرِّهِ **الْكَسْرَةُ** (مَمْنُوعٌ مِنَ الصَّرْفِ لَكِنْ هُنَا مُضَافٌ → الْكَسْرَةُ تَرْجِعُ !)
> - **الْمَدِينَةِ** : مُضَافٌ إِلَيْهِ مَجْرُورٌ وَعَلَامَةُ جَرِّهِ الْكَسْرَةُ

## Partie 18 — Traduction libre `/10`

Traduis en arabe :

| # | Français |
|---|---|
| 1 | L'étudiant a écrit la leçon avec le stylo. |
| 2 | As-tu (m.) un frère ? — Oui, j'ai un frère. |
| 3 | Ce livre est à Fatima. |
| 4 | Je suis allé à la mosquée avec mon ami. |
| 5 | Les deux étudiantes sont parties avant le professeur. |

> [!tip]- Correction
> 1. كَتَبَ الطَّالِبُ الدَّرْسَ **بِالْقَلَمِ**.
> 2. هَلْ **عِنْدَكَ** أَخٌ ؟ — نَعَمْ، **عِنْدِي** أَخٌ.
> 3. هَذَا الْكِتَابُ **لِفَاطِمَةَ**. (لِـ = propriété, فَاطِمَةَ = مَمْنُوعٌ مِنَ الصَّرْفِ → فَتْحَةٌ)
> 4. ذَهَبْتُ **إِلَى** الْمَسْجِدِ **مَعَ** صَدِيقِي.
> 5. ذَهَبَتِ **الطَّالِبَتَانِ** **قَبْلَ** الْمُعَلِّمِ. (duel féminin)
