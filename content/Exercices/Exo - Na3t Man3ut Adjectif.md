## Exercice 1 — النعت ou الخبر ?

C'est le piège principal ! Dis si c'est un **نعت** (adjectif) ou un **خبر** (prédicat = phrase nominale) :

| # | Phrase | نعت ou خبر ? | Traduction |
|---|---|---|---|
| 1 | الطالبُ **المجتهدُ** | | |
| 2 | الطالبُ **مجتهدٌ** | | |
| 3 | كتابٌ **جديدٌ** | | |
| 4 | الكتابُ **جديدٌ** | | |
| 5 | البيتُ **الكبيرُ** | | |
| 6 | البيتُ **كبيرٌ** | | |

> [!tip]- Correction
> 1. **نعت** — معرفة + معرفة = adjectif → "l'élève studieux"
> 2. **خبر** — معرفة + نكرة = phrase nominale → "l'élève **EST** studieux"
> 3. **نعت** — نكرة + نكرة = adjectif → "un livre neuf"
> 4. **خبر** — معرفة + نكرة = phrase nominale → "le livre **EST** neuf"
> 5. **نعت** — معرفة + معرفة = adjectif → "la grande maison"
> 6. **خبر** — معرفة + نكرة = phrase nominale → "la maison **EST** grande"
>
> **Règle :** si les deux sont du même type (معرفة+معرفة ou نكرة+نكرة) → نعت. Si c'est معرفة+نكرة → خبر !

## Exercice 2 — Accorde l'adjectif

Le نعت doit s'accorder en **4 choses**. Complète :

| # | Phrase | Adjectif accordé |
|---|---|---|
| 1 | الطالبُ ال... (مجتهد) | |
| 2 | الطالبةُ ال... (مجتهد) | |
| 3 | طالبٌ ... (مجتهد) | |
| 4 | في المدرسةِ ال... (كبير) | |
| 5 | رأيتُ الولدَ ال... (صغير) | |
| 6 | طالبانِ ... (مجتهد) | |

> [!tip]- Correction
> 1. الطالبُ **المجتهدُ** — معرفة + مذكر + مرفوع
> 2. الطالبةُ **المجتهدةُ** — معرفة + مؤنث (ajout ة) + مرفوع
> 3. طالبٌ **مجتهدٌ** — نكرة + مذكر + مرفوع (tanwīn car نكرة)
> 4. في المدرسةِ **الكبيرةِ** — معرفة + مؤنث + مجرور (après في)
> 5. رأيتُ الولدَ **الصغيرَ** — معرفة + مذكر + منصوب
> 6. طالبانِ **مجتهدانِ** — نكرة + مذكر + مثنى مرفوع

## Exercice 3 — Les 4 accords

Pour chaque phrase, vérifie les 4 accords (الإعراب, التعريف, الجنس, العدد) :

| Phrase | إعراب | تعريف | جنس | عدد | Correct ? |
|---|---|---|---|---|---|
| الطالبُ الجديدةُ | مرفوع ✅ | معرفة ✅ | ❓ | ❓ | |
| كتابٌ الكبيرُ | ❓ | ❓ | ❓ | ❓ | |
| في البيتِ الكبيرِ | ❓ | ❓ | ❓ | ❓ | |

> [!tip]- Correction
> 1. الطالبُ الجديد**ةُ** → **FAUX** — الجنس ne correspond pas ! طالب = مذكر mais جديدة = مؤنث. Correct : الطالبُ الجديدُ
> 2. كتابٌ **ال**كبيرُ → **FAUX** — التعريف ne correspond pas ! كتابٌ = نكرة mais الكبير = معرفة. Correct : كتابٌ كبيرٌ
> 3. في البيتِ الكبيرِ → **CORRECT** — مجرور ✅ معرفة ✅ مذكر ✅ مفرد ✅

## Exercice 4 — Construis des groupes نعت + منعوت

Combine les éléments :

| # | Nom | Adjectif | Cas | Ta réponse |
|---|---|---|---|---|
| 1 | مدرسة + معرفة | كبير | مرفوع | |
| 2 | كتاب + نكرة | جديد | منصوب | |
| 3 | طالب + معرفة | صغير | مجرور | |

> [!tip]- Correction
> 1. **المدرسةُ الكبيرةُ** (معرفة + مؤنث + مرفوع)
> 2. **كتابًا جديدًا** (نكرة + مذكر + منصوب avec tanwīn)
> 3. **الطالبِ الصغيرِ** (معرفة + مذكر + مجرور)
