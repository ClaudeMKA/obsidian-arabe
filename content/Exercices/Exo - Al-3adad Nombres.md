## Exercice 1 — La règle d'inversion du genre

Rappel : le nombre est **OPPOSÉ** en genre au nom compté !

Complète avec la bonne forme du nombre :

| # | Nombre + Nom | Ta réponse |
|---|---|---|
| 1 | 3 + طلاب (طالب = masc.) | |
| 2 | 3 + طالبات (طالبة = fém.) | |
| 3 | 5 + كتب (كتاب = masc.) | |
| 4 | 5 + ساعات (ساعة = fém.) | |
| 5 | 7 + رجال (رجل = masc.) | |
| 6 | 4 + بنات (بنت = fém.) | |

> [!tip]- Correction
> 1. **ثلاثةُ** طلابٍ — طالب = مذكر → nombre avec **ة**
> 2. **ثلاثُ** طالباتٍ — طالبة = مؤنث → nombre **sans ة**
> 3. **خمسةُ** كتبٍ — كتاب = مذكر → avec **ة**
> 4. **خمسُ** ساعاتٍ — ساعة = مؤنث → sans **ة**
> 5. **سبعةُ** رجالٍ — رجل = مذكر → avec **ة**
> 6. **أربعُ** بناتٍ — بنت = مؤنث → sans **ة**

## Exercice 2 — Quel est le genre du singulier ?

Attention : on regarde le **singulier**, pas le pluriel !

| # | Pluriel | Singulier | Genre | Le nombre 5 sera... |
|---|---|---|---|---|
| 1 | كتبٌ | كتابٌ | | |
| 2 | طالباتٌ | طالبةٌ | | |
| 3 | أيامٌ | يومٌ | | |
| 4 | ليالٍ | ليلةٌ | | |
| 5 | أقلامٌ | قلمٌ | | |

> [!tip]- Correction
> 1. كتابٌ = **مذكر** → خمس**ةُ** كتبٍ (avec ة)
> 2. طالبةٌ = **مؤنث** → خمس**ُ** طالباتٍ (sans ة)
> 3. يومٌ = **مذكر** → خمس**ةُ** أيامٍ (avec ة)
> 4. ليلةٌ = **مؤنث** → خمس**ُ** ليالٍ (sans ة)
> 5. قلمٌ = **مذكر** → خمس**ةُ** أقلامٍ (avec ة)

## Exercice 3 — Le معدود est جمع مجرور

Complète avec le bon إعراب du nom compté :

| # | Phrase | Correct ? |
|---|---|---|
| 1 | ثلاثةُ كتبٌ | |
| 2 | ثلاثةُ كتبٍ | |
| 3 | أربعُ طالباتٌ | |
| 4 | أربعُ طالباتٍ | |

> [!tip]- Correction
> 1. **FAUX** — le معدود doit être **مجرور** (مضاف إليه)
> 2. **CORRECT** — ثلاثةُ كتبٍ (كتبٍ = جمع مجرور بالكسرة)
> 3. **FAUX** — même raison
> 4. **CORRECT** — أربعُ طالباتٍ (طالباتٍ = جمع مجرور)

## Exercice 4 — Traduis avec les nombres

| # | Français | Arabe |
|---|---|---|
| 1 | 3 livres | |
| 2 | 6 filles | |
| 3 | 10 hommes | |
| 4 | 8 heures | |
| 5 | 4 stylos | |

> [!tip]- Correction
> 1. **ثلاثةُ** كتبٍ
> 2. **ستُّ** بناتٍ (بنت = مؤنث → sans ة)
> 3. **عشرةُ** رجالٍ (رجل = مذكر → avec ة)
> 4. **ثمانِي** ساعاتٍ (ساعة = مؤنث → sans ة)
> 5. **أربعةُ** أقلامٍ (قلم = مذكر → avec ة)
