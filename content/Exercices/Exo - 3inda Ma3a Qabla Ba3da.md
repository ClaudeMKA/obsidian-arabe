## Exercice 2 — مَعَ ou بـ ?

Rappel : **مَعَ** = accompagnement (personnes), **بـ** = instrument (outils)

| # | Phrase | مَعَ ou بـ ? |
|---|---|---|
| 1 | Je suis allé ... Mohammed | |
| 2 | J'ai écrit ... le stylo | |
| 3 | Elle est venue ... sa sœur | |
| 4 | Il a coupé ... le couteau | |
| 5 | Nous sommes partis ... (ensemble) | |

> [!tip]- Correction
> 1. ذهبتُ **مَعَ** محمدٍ (accompagnement)
> 2. كتبتُ **بـ**القلمِ (instrument)
> 3. جاءَتْ **مَعَ** أختِها (accompagnement)
> 4. قطعَ **بـ**السكينِ (instrument)
> 5. ذهبنا **مَعًا** (ensemble — avec tanwīn fatha !)

## Exercice 4 — Le grand piège : ظرف ou حرف جر ?

| # | Mot | ظرف ou حرف جر ? |
|---|---|---|
| 1 | في | |
| 2 | عِندَ | |
| 3 | على | |
| 4 | مَعَ | |
| 5 | من | |
| 6 | قَبْلَ | |
| 7 | إلى | |
| 8 | بَعْدَ | |

> [!tip]- Correction
> 1. في → **حرف جر**
> 2. عِندَ → **ظرف مكان** (PAS un حرف جر !)
> 3. على → **حرف جر**
> 4. مَعَ → **ظرف مكان** (PAS un حرف جر !)
> 5. من → **حرف جر**
> 6. قَبْلَ → **ظرف زمان** (PAS un حرف جر !)
> 7. إلى → **حرف جر**
> 8. بَعْدَ → **ظرف زمان** (PAS un حرف جر !)
>
> **Astuce :** عِندَ, مَعَ, قَبْلَ, بَعْدَ sont des **ظروف**. Le mot après est **مضاف إليه مجرور** (pas مجرور par un حرف جر, mais par الإضافة).

## Exercice 6 — عِندَ : question → réponse

Réponds aux questions en utilisant عِندَ :

| # | Question | Réponse |
|---|---|---|
| 1 | هل **عِندَكَ** كتابٌ ؟ | نعم، ... (oui, j'ai un livre) |
| 2 | هل **عِندَها** سيارةٌ ؟ | لا، ... (non, elle n'a pas de voiture) |
| 3 | هل **عِندَكُم** أولادٌ ؟ | نعم، ... ثلاثةُ أولادٍ (oui, nous avons 3 enfants) |
| 4 | **ماذا عِندَكَ** ؟ | ... كتابٌ وقلمٌ (j'ai un livre et un stylo) |
| 5 | هل **عِندَهُ** إخوةٌ ؟ | لا، ... (non, il n'a pas de frères) |
| 6 | **كم كتابًا عِندَكَ** ؟ | ... خمسةُ كتبٍ (j'ai 5 livres) |

> [!tip]- Correction
> 1. نعم، **عِندي** كتابٌ
> 2. لا، **ما عِندَها** سيارةٌ (ou ليسَ عِندَها سيارةٌ)
> 3. نعم، **عِندَنا** ثلاثةُ أولادٍ
> 4. **عِندي** كتابٌ وقلمٌ
> 5. لا، **ما عِندَهُ** إخوةٌ (ou ليسَ عِندَهُ إخوةٌ)
> 6. **عِندي** خمسةُ كتبٍ

## Exercice 7 — عِندَ ou لِـ ?

Rappel : **عِندَ** = possession physique (c'est chez moi) / **لِـ** = propriété (ça m'appartient)

Choisis la bonne réponse :

| # | Phrase en français | عِندَ ou لِـ ? | En arabe |
|---|---|---|---|
| 1 | J'ai un livre (il est chez moi) | | |
| 2 | Ce livre est à Mohammed (propriété) | | |
| 3 | Elle a un stylo (sur elle) | | |
| 4 | La voiture est à mon père | | |
| 5 | Nous avons un examen demain | | |
| 6 | À qui est ce sac ? | | |
| 7 | As-tu des frères ? | | |
| 8 | La maison est à nous | | |

> [!tip]- Correction
> 1. **عِندي** كتابٌ → **عِندَ** (le livre est physiquement chez moi)
> 2. الكتابُ **لِمحمدٍ** → **لِـ** (propriété)
> 3. **عِندَها** قلمٌ → **عِندَ** (le stylo est avec elle)
> 4. السيارةُ **لِأبي** → **لِـ** (propriété)
> 5. **عِندَنا** امتحانٌ غدًا → **عِندَ** (situation)
> 6. **لِمَن** هذه الحقيبةُ ؟ → **لِـ** (on demande la propriété)
> 7. هل **عِندَكَ** إخوةٌ ؟ → **عِندَ** (les deux marchent, mais عِندَ est plus courant ici)
> 8. البيتُ **لَنا** → **لِـ** (propriété)
>
> **Astuce :** En pratique, les deux sont souvent interchangeables. Mais si tu demandes **à qui** appartient quelque chose → **لِـ**. Si tu dis que quelque chose **est chez** quelqu'un → **عِندَ**.

## Exercice 8 — عِندَ : lieu ou possession ?

Indique si عِندَ a le sens de **lieu** (chez/auprès de) ou de **possession** (avoir) :

| # | Phrase | Lieu ou Possession ? |
|---|---|---|
| 1 | عِندي كتابٌ | |
| 2 | أنا عِندَ المعلمِ | |
| 3 | عِندَهُ سيارةٌ جديدةٌ | |
| 4 | جلستُ عِندَ البابِ | |
| 5 | ما عِندَنا وقتٌ | |
| 6 | ذهبتُ عِندَ صديقي | |
| 7 | هل عِندَكَ أولادٌ ؟ | |
| 8 | الكتابُ عِندَ محمدٍ | |

> [!tip]- Correction
> 1. **Possession** — J'ai un livre
> 2. **Lieu** — Je suis chez le professeur
> 3. **Possession** — Il a une voiture neuve
> 4. **Lieu** — Je me suis assis près de la porte
> 5. **Possession** — Nous n'avons pas le temps
> 6. **Lieu** — Je suis allé chez mon ami
> 7. **Possession** — As-tu des enfants ?
> 8. **Les deux !** — Le livre est chez Mohammed (lieu) = Mohammed a le livre (possession)

## Exercice 9 — عِندَ : traduis le dialogue

Traduis ce dialogue en arabe en utilisant عِندَ :

> — As-tu (m.) un stylo ?
> — Non, je n'ai pas de stylo.
> — Et Mohammed, il a un stylo ?
> — Oui, il a un stylo rouge.
> — Le stylo est chez Mohammed ?
> — Oui, le stylo est chez lui.

> [!tip]- Correction
> — هل **عِندَكَ** قلمٌ ؟
> — لا، **ما عِندي** قلمٌ.
> — ومحمدٌ، هل **عِندَهُ** قلمٌ ؟
> — نعم، **عِندَهُ** قلمٌ أحمرُ.
> — القلمُ **عِندَ** محمدٍ ؟
> — نعم، القلمُ **عِندَهُ**.

## Exercice 10 — عِندَ : les pièges de l'إعراب

Quel est le إعراب du mot **après** عِندَ ? Rappel : عِندَ est un ظرف مضاف, donc le mot après est مضاف إليه مجرور.

| # | Phrase | Mot après عِندَ | إعراب |
|---|---|---|---|
| 1 | عِندَ المعلمِ | المعلمِ | |
| 2 | عِندَ محمدٍ | محمدٍ | |
| 3 | عِندَ فاطمةَ | فاطمةَ | |
| 4 | عِندَ الطالبَيْنِ | الطالبَيْنِ | |
| 5 | عِندَ المسلمِينَ | المسلمِينَ | |

> [!tip]- Correction
> Tous sont **مضاف إليه مجرور** :
> 1. المعلمِ → مجرور بال**كسرة** (اسم مفرد معرفة)
> 2. محمدٍ → مجرور بال**كسرة** + تنوين (عَلَم ينصرف)
> 3. فاطمةَ → مجرور بال**فتحة** (ممنوع من الصرف ! → voir [[Mamnu min Sarf - Interdit de tanwin]])
> 4. الطالبَيْنِ → مجرور بال**ياء** (مثنى → voir [[Muthanna - Le duel]])
> 5. المسلمِينَ → مجرور بال**ياء** (جمع مذكر سالم)
>
> **Piège :** فاطمةَ prend une **فتحة** au lieu d'une كسرة car c'est un nom ممنوع من الصرف !
