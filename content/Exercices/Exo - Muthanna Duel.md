## Exercice 1 — Forme le duel

Mets chaque mot au duel (المثنى) au cas مرفوع :

| # | Singulier | Duel مرفوع |
|---|---|---|
| 1 | رجلٌ | |
| 2 | كتابٌ | |
| 3 | طالبةٌ | |
| 4 | سيارةٌ | |
| 5 | معلمٌ | |
| 6 | غرفةٌ | |

> [!tip]- Correction
> 1. رجل**انِ**
> 2. كتاب**انِ**
> 3. طالبت**انِ** (ة → ت avant انِ)
> 4. سيارت**انِ** (ة → ت)
> 5. معلم**انِ**
> 6. غرفت**انِ** (ة → ت)

## Exercice 2 — انِ ou يْنِ ?

Choisis la bonne terminaison selon le إعراب :

| # | Phrase | Terminaison | Pourquoi ? |
|---|---|---|---|
| 1 | جاءَ الطالب... | | |
| 2 | رأيتُ الطالب... | | |
| 3 | سلّمتُ على الطالب... | | |
| 4 | هذ... قلم... جديد... | | |
| 5 | اشتريتُ كتاب... | | |

> [!tip]- Correction
> 1. الطالب**انِ** — مرفوع (فاعل) → انِ
> 2. الطالب**َيْنِ** — منصوب (مفعول به) → يْنِ
> 3. الطالب**َيْنِ** — مجرور (بعد على) → يْنِ
> 4. **هذانِ** قلم**انِ** جديد**انِ** — tout مرفوع → انِ
> 5. كتاب**َيْنِ** — منصوب → يْنِ

## Exercice 3 — Féminin au duel

Attention au **ة** qui devient **ت** ! Mets au duel :

| # | Singulier | Duel (مرفوع) | Duel (منصوب/مجرور) |
|---|---|---|---|
| 1 | مدرسةٌ (école) | | |
| 2 | طاولةٌ (table) | | |
| 3 | معلمةٌ (enseignante) | | |

> [!tip]- Correction
> 1. مدرست**انِ** / مدرست**َيْنِ**
> 2. طاولت**انِ** / طاولت**َيْنِ**
> 3. معلمت**انِ** / معلمت**َيْنِ**
