## Exercice 1 — Choisis le bon démonstratif

Complète avec **هذا / هذه / هذانِ / هاتانِ / هؤلاءِ** :

| # | Phrase | Démonstratif |
|---|---|---|
| 1 | ... كتابٌ | |
| 2 | ... سيارةٌ | |
| 3 | ... رجلانِ | |
| 4 | ... طالبتانِ | |
| 5 | ... رجالٌ | |
| 6 | ... بنتٌ | |
| 7 | ... قلمٌ | |
| 8 | ... كتبٌ | |

> [!tip]- Correction
> 1. **هذا** كتابٌ (masc. sing.)
> 2. **هذه** سيارةٌ (fém. sing.)
> 3. **هذانِ** رجلانِ (duel masc.)
> 4. **هاتانِ** طالبتانِ (duel fém.)
> 5. **هؤلاءِ** رجالٌ (pluriel عاقل)
> 6. **هذه** بنتٌ (fém. sing.)
> 7. **هذا** قلمٌ (masc. sing.)
> 8. **هذه** كتبٌ (pluriel غير عاقل = traité comme **مفرد مؤنث** !)

## Exercice 2 — Proche ou loin ?

Traduis avec le bon démonstratif :

| # | Français | Arabe |
|---|---|---|
| 1 | Ce livre | |
| 2 | Cette voiture là-bas | |
| 3 | Cet homme là-bas | |
| 4 | Ces femmes (proches) | |
| 5 | Ces deux stylos (proches) | |

> [!tip]- Correction
> 1. **هذا** الكتابُ
> 2. **تلك** السيارةُ
> 3. **ذلك** الرجلُ
> 4. **هؤلاءِ** النساءُ
> 5. **هذانِ** القلمانِ

## Exercice 3 — Le grand piège : عاقل ou غير عاقل ?

Choisis entre **هؤلاءِ** et **هذه** pour le pluriel :

| # | Mot | هؤلاءِ ou هذه ? | Pourquoi ? |
|---|---|---|---|
| 1 | طلابٌ (étudiants) | | |
| 2 | سياراتٌ (voitures) | | |
| 3 | معلمونَ (enseignants) | | |
| 4 | أقلامٌ (stylos) | | |
| 5 | بناتٌ (filles) | | |
| 6 | كتبٌ (livres) | | |
| 7 | أطباءُ (médecins) | | |
| 8 | مدارسُ (écoles) | | |

> [!tip]- Correction
> 1. **هؤلاءِ** طلابٌ — عاقل (humains)
> 2. **هذه** سياراتٌ — غير عاقل (choses → مفرد مؤنث)
> 3. **هؤلاءِ** معلمونَ — عاقل
> 4. **هذه** أقلامٌ — غير عاقل
> 5. **هؤلاءِ** بناتٌ — عاقل
> 6. **هذه** كتبٌ — غير عاقل
> 7. **هؤلاءِ** أطباءُ — عاقل
> 8. **هذه** مدارسُ — غير عاقل
