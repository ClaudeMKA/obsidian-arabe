Cours : [[Damaair - Les pronoms]]

## Exercice 1 — Quel pronom séparé ?

Traduis en arabe avec le bon **ضمير منفصل** :

| # | Français | الضمير المنفصل |
|---|---|---|
| 1 | Je suis étudiant | ... طالبٌ |
| 2 | Nous sommes musulmans | ... مسلمونَ |
| 3 | Tu es intelligent (à un garçon) | ... ذكيٌّ |
| 4 | Tu es intelligente (à une fille) | ... ذكيةٌ |
| 5 | Il est médecin | ... طبيبٌ |
| 6 | Elle est enseignante | ... مُدرِّسةٌ |
| 7 | Vous deux êtes étudiants | ... طالبانِ |
| 8 | Ils sont grands | ... كبارٌ |

> [!tip]- Correction
> 1. **أنا** طالبٌ
> 2. **نحنُ** مسلمونَ
> 3. **أنتَ** ذكيٌّ
> 4. **أنتِ** ذكيةٌ
> 5. **هُوَ** طبيبٌ
> 6. **هِيَ** مُدرِّسةٌ
> 7. **أنتُما** طالبانِ
> 8. **هُم** كبارٌ

## Exercice 2 — Complète avec le pronom attaché

Ajoute le bon **ضمير متصل** :

| # | Phrase à compléter | Sens |
|---|---|---|
| 1 | كتاب... | mon livre |
| 2 | كتاب... | ton livre (masc.) |
| 3 | كتاب... | ton livre (fém.) |
| 4 | كتاب... | son livre (à lui) |
| 5 | كتاب... | son livre (à elle) |
| 6 | كتاب... | notre livre |
| 7 | كتاب... | votre livre (masc. pl.) |
| 8 | كتاب... | leur livre (masc.) |
| 9 | علّمَ... | il m'a enseigné |
| 10 | علّمَ... | il t'a enseigné (masc.) |

> [!tip]- Correction
> 1. كتاب**ي** (ـي)
> 2. كتاب**ُكَ** (ـكَ)
> 3. كتاب**ُكِ** (ـكِ)
> 4. كتاب**ُهُ** (ـهُ)
> 5. كتاب**ُها** (ـها)
> 6. كتاب**ُنا** (ـنا)
> 7. كتاب**ُكُم** (ـكُم)
> 8. كتاب**ُهُم** (ـهُم)
> 9. علّمَ**ني** (ـني)
> 10. علّمَ**كَ** (ـكَ)

## Exercice 3 — Piège ! Quel pronom pour le pluriel non-rationnel ?

Choisis le bon pronom pour remplacer le mot en gras :

| # | Phrase | هُم ou هِيَ ? |
|---|---|---|
| 1 | **الرجالُ** جاؤوا | |
| 2 | **الكتبُ** على الطاولةِ | |
| 3 | **السياراتُ** جديدةٌ | |
| 4 | **الطالباتُ** مجتهداتٌ | |
| 5 | **الأقلامُ** كثيرةٌ | |

> [!tip]- Correction
> 1. **هُم** جاؤوا — رجال = عاقل (humains) → هُم
> 2. **هِيَ** على الطاولةِ — كتب = غير عاقل (choses) → **هِيَ** (on traite comme مفرد مؤنث)
> 3. **هِيَ** جديدةٌ — سيارات = غير عاقل → **هِيَ**
> 4. **هُنَّ** مجتهداتٌ — طالبات = عاقل féminin → هُنَّ
> 5. **هِيَ** كثيرةٌ — أقلام = غير عاقل → **هِيَ**
